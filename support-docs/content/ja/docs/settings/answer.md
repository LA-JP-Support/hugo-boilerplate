## 事前回答（Predefined answers）とは

よくある質問や定型的な問い合わせに対して、あらかじめ準備した回答をすぐに呼び出して送信できる機能です。回答の一貫性を保ち、応答時間を短縮できます。

### 具体的な活用例

例えばFAQで頻出する「パスワードリセット方法」や「配送状況確認方法」を登録しておけば、エージェントは数秒で回答を送信できます。チーム全体で統一したメッセージを利用できるため、ブランドイメージや信頼性を高められます。

## 主な利点

### 迅速な顧客対応
- よくある質問に即座に対応できる

### 回答品質の向上
- 回答の一貫性を確保
- チーム全体で標準化した回答を利用可能

### カスタマーサービスの効率化
- 応答時間を短縮して顧客満足度を向上
- ブランドイメージを守る効果がある
