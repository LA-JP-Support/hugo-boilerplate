## コンタクトフォームとは

顧客からの問い合わせを直接チケット化できるWebフォームです。

| 特徴 | 説明 |
|------|------|
| **自動チケット化** | 顧客の入力を即座にチケットとして登録 |
| **セキュリティ** | メールアドレスや電話番号を公開せずに受付可能 |
| **24時間対応** | いつでも問い合わせを受け付け |
| **FAQ連携** | 入力中に関連FAQを自動サジェスト |

## フォームの種類

| 種類 | 説明 | 用途 |
|------|------|------|
| **問合せボタン** | ボタンクリックでページ内にフォーム表示 | サイト全体での問い合わせ受付 |
| **ページフォーム** | 問い合わせ専用ページを作成 | 詳細な情報収集が必要な場合 |

## フォームギャラリー

あらかじめ用意されたテンプレートから選ぶだけで、すぐに利用開始できます。

### ギャラリーのメリット

| メリット | 説明 |
|----------|------|
| **簡単導入** | テンプレートを選ぶだけ |
| **ブランディング** | デザイン・形式を統一 |
| **複数窓口** | サポート用、営業用など目的別に設置可能 |

## 設定手順

### 1. ボタンの作成

「設定」→「コンタクトフォーム」→「作成」から新規フォームを作成します。

- カラースタイルの選択
- オリジナル画像でのカスタマイズ
- 言語設定

### 2. フォームの設定

| 設定項目 | 内容 |
|----------|------|
| **タイトル** | フォームの見出し |
| **ロゴ** | 企業ロゴの表示 |
| **FAQ表示** | ナレッジベースとの連携 |
| **送信後メッセージ** | 完了時の表示内容 |

### 3. 入力項目の設定

| 設定項目 | 内容 |
|----------|------|
| **チーム選択** | 問い合わせ先の部門選択 |
| **カスタムフィールド** | 追加の入力項目 |
| **必須項目** | 入力必須の指定 |

### 4. デザインのカスタマイズ

- スタイルと色の調整
- サイトのデザインに合わせた色調設定

### 5. Webサイトへの設置

生成されたコードを、Webサイトの `</body>` タグ前に配置します。

```html
<!-- LiveAgent コンタクトフォーム -->
<script>...</script>
```

## 注意事項

| 項目 | 内容 |
|------|------|
| **ローカルファイル** | 表示不可（Webサーバーが必要） |
| **SSL対応** | HTTPS環境では専用設定が必要 |
