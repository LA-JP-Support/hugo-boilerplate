## 「解決ボタン」機能の概要

### 機能の目的
SLA違反が近づいているチケットを強調表示し、迅速な対応を促す機能です。エージェントは優先すべき案件を一目で把握できます。

### 使用方法
例えば「解決ボタン」をクリックすると、期限が迫ったチケットを一覧で確認でき、効率的に優先対応が可能です。

## 主な特徴

### 通知機能
- SLA違反が近いチケットを通知

### 作業効率の向上
- 優先案件を一覧で確認可能
- 対応の遅れを防止

### メリット
- 顧客満足度を維持
- エージェントの作業効率を向上
