## 翻訳のカスタマイズ方法

言語ファイルの翻訳を別の言葉に置き換えて、独自の翻訳に変更することができます。

例として、カスタマーポータルページに表示される「チケットを提出」を「新規お問い合わせ」に変更する手順をご案内します。

![画像1](/liveagent/scripts/file.php?view=Y&file=056c1c978c26a5400f1ec231e9e09abc)

### 翻訳変更の手順

#### 言語設定画面への移動

1)「設定」→「言語」で言語リストを表示し、「日本語」の行で「編集」をクリック

![画像2](/liveagent/scripts/file.php?view=Y&file=a0b75da534e6a7e75d5daa63ebc70f2d)

#### 翻訳対象の選択

2)「言語を編集」の画面で、①「翻訳」タブを選択 → ②変更したい文字列（チケットを提出）を入力 → ③該当するソースメッセージ（Submit ticket）上をクリック

![画像3](/liveagent/scripts/file.php?view=Y&file=e5f2943c178f2d19e8ae495142bdee11)

#### 翻訳の編集と保存

3)「翻訳を編集」の画面で、「チケットを提出」という翻訳を「新規お問い合わせ」に書き換えて「保存」

![画像4](/liveagent/scripts/file.php?view=Y&file=c66bc4e57ccc6a199edec764cc75005b)

#### 言語設定の確定

4)画面を閉じると言語のリストページに戻るので、上部にある「保存」ボタンをクリック

![画像5](/liveagent/scripts/file.php?view=Y&file=267683ab7244ba0826c0c9f128197edd)

### アプリケーションの再起動

5)次に表示される画面でアプリケーションを再起動。再起動後に変更が反映されます。

![画像6](/liveagent/scripts/file.php?view=Y&file=b30c54335c6c9fd718c32341dfce3a66)
