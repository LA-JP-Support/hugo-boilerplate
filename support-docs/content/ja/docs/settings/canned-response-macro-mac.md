## 定型文・マクロとは

頻繁に使う回答をテンプレート化してすぐに挿入できる機能です。回答の一貫性を保ち、対応スピードを向上させます。

例えば「パスワードリセットの方法」といったよくある質問に対し、定型文を用意しておけば数秒で回答できます。マクロとして複数操作を一括実行することも可能です。

## 主な特徴

### テンプレート化のメリット

- よく使う回答をテンプレート化
- ワンクリックで回答を挿入できる
- 回答の一貫性とスピードを向上

### 機能の拡張性

- マクロで複数操作を自動化可能
- エージェントの負担を軽減
