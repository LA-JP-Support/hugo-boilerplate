## LiveAgent API 概要

LiveAgent では、業務に役立つ API を提供しています。開発者の方々に向けて、各種 API のドキュメントおよびサンプルコードを紹介します。

## API の基本情報

### データフィールドのフォーマット

API で使用されるデータフィールドの種類と形式を詳細に説明します。

- **datetime**: 日付・時刻のフォーマット
- **text**: 文字列
- **constlist**: 固定値リスト
- **int**: 整数値
- **list**: エンティティのリスト
- **opdatetime**: 日付演算子

### データ出力形式

#### サポートされる出力形式
- JSON（デフォルト）
- XML
- YAML（PHP YAML 拡張が有効な場合）

## API エンドポイントと機能

### REST API v.1 の主な機能

#### エージェント関連
- エージェント情報の取得
- 認証トークンの管理

#### アプリケーション関連
- インストール日時の参照
- バージョン情報の確認

#### 会話（チケット）管理
- チケットの転送
- チケットの削除
- チケット情報の参照
- メッセージの追加
- チケットステータスの変更
- タグ管理

#### 顧客情報管理
- 顧客情報の取得
- 顧客グループの管理
- 新規顧客登録

#### その他の機能
- チーム管理
- ナレッジベース
- タグ管理

## エラーハンドリング

### HTTP エラーコード

#### クライアントエラー
- **400 - Bad Request**: API キー認証エラー
- **401 - Forbidden**: 不正な API キー
- **404 - Not Found**: リソース未検出

#### サーバーエラー
- **500 - 一般的な処理エラー**: パラメータ不備
- **503 - Service Unavailable**: サービス無効
