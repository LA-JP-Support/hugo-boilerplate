## 多言語サポート機能の概要
### 多言語サポート機能とは
**多言語サポート機能**&#8203;とは、グローバルに展開する企業や複数の言語で顧客サービスを提供する企業が、カスタマーサポートエージェント（担当者）に好みの言語でLiveAgentを使用させることができる機能です。

LiveAgentは現在**43の言語**&#8203;をサポートしており、新しい翻訳が定期的に追加されています。

## 対応言語の範囲
### サポート言語の多様性
LiveAgentは世界43言語に対応し、主要な国際言語から地域特有の言語まで幅広くカバーしています。これにより、世界中の多様な市場で事業を展開する企業のニーズに対応できます。

## 多言語サポートの主要機能
### ログイン言語の設定
エージェントが好みの言語でLiveAgentにログインし、操作できます。これにより：

- **直感的な操作**: 母国語での操作により作業効率が向上
- **学習コストの削減**: 新人研修時の言語的障壁を軽減
- **ストレス軽減**: 慣れ親しんだ言語での作業による心理的負担の軽減

### 言語適応ウィジェット
LiveAgentでウィジェットをカスタマイズ・作成する際に、特定の部門に対して好みの言語を簡単に設定できます。

**活用例：**

- 異なる言語の市場で事業を展開している場合、ウェブサイトウィジェットも適応させることが可能
- 日本語市場向けには日本語ウィジェット、英語圏向けには英語ウィジェットを設置
- 地域特有の文化的ニュアンスに合わせたカスタマイズが可能

### 継続的な言語サポート改善

- **コミュニティ貢献**: ユーザーからの翻訳協力による言語サポート拡大

## まとめ：グローバルビジネスの言語的障壁解消
LiveAgentの多言語サポート機能は、単なる翻訳機能を超えて、グローバルビジネスにおける「言語的公平性」と「文化的適応性」を実現する戦略的ツールです。

**主な価値：**

- **グローバル統一**: 世界各地で一貫したサービス品質を提供
- **現地化戦略**: 各市場の言語・文化に最適化されたカスタマーエクスペリエンス
- **運営効率化**: 単一プラットフォームでの多言語運用によるコスト削減
- **人材活用**: 多様な言語的背景を持つスタッフの能力最大化
- **拡張性**: 新市場参入時の迅速かつ効率的な対応

43言語という豊富な対応言語数と、継続的な言語追加により、LiveAgentは真のグローバル企業のパートナーとして、言語の壁を越えた優れた顧客サービスの実現を支援します。
