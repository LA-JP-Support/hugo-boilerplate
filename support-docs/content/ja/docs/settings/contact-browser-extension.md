## 概要

Web Contact Cardsは、LiveAgentのブラウザ拡張機能で、ウェブ上のメールアドレスや電話番号にマウスを重ねると、その連絡先に関する情報をポップアップ表示します。

## 表示される情報

- 連絡先のアバター/プロフィール画像
- メールアドレス
- 電話番号
- 役職
- その連絡先に関連するチケット総数
- 未解決チケット数

## 主な特長

### 作業効率の向上

- ウェブからデータをコピー&ペーストする手間を省略
- メールアドレスをクリックするだけで事前入力済みチケットを作成

### 柔軟な活用

- メール対応とコールセンター業務の両方で使用可能
- 電話番号をクリックしてブラウザ内通話を開始

## 対応環境

### 対応ブラウザ

- Google Chrome
- Mozilla Firefox

### 代替方法

ブラウザ拡張機能を使用しない場合は、JavaScriptコードを内部CRMに配置することも可能

## 動作仕組み

- `mailto:` と `tel:` リンクにマウスを重ねると自動的に情報を表示
- ウェブ全体または特定のウェブサイトのみで動作するよう設定可能

## 入手方法

- Google Chrome Web Store
- Firefox Browser Add-ons

## まとめ

この機能により、顧客情報の確認と対応開始がスムーズになり、よりパーソナライズされた顧客サービスを提供できます。
