## 動作環境（エンドユーザ側）

LiveAgentの機能を組み込んだWebページを閲覧できる環境は下記のとおりです。最新のコミュニケーション機能を安全かつスムーズに利用いただくため、推奨される環境をご確認ください。

### パソコン（PC）環境

#### サポートOS
- Windows：Windows 10以降の最新バージョン推奨
- Mac OS：最新の macOS
- Linux：Ubuntu、Fedoraなどの主要ディストリビューション

#### サポートブラウザ
- Edge：最新バージョン推奨
- Chrome：最新バージョン
- Safari：最新バージョン
- Firefox：最新バージョン

LiveAgentは最新のWeb技術を活用しているため、常に最新のブラウザでの使用をお勧めします。古いブラウザでは、チャット機能や画面共有などの高度な機能が正常に動作しない可能性があります。

**注意:** Internet Explorerのサポートは終了しました。チャットウィジェット等は正常に動作しません。

### モバイル環境（スマートフォン/タブレット）

#### サポートOS
- iOS：iOS 13以降
- Android：Android 9.0以降

#### サポートブラウザ
- Safari：iOSの標準ブラウザ
- Chrome：Androidの標準ブラウザ
- Firefox：最新モバイル版

モバイル環境では、画面サイズや通信状況に応じて、レスポンシブデザインで最適化されたインターフェースを提供します。スマートフォンやタブレットからでも、PC同様の快適な操作性を実現しています。

通信環境については、安定したインターネット接続（Wi-Fiまたは4G/5G）を推奨します。低速または不安定な回線では、チャット機能や画面共有の品質に影響が出る可能性があります。
