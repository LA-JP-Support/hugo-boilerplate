## トラブル時のサポート体制

### サポート対応の特徴
SmartWebでは、開発元と弊社による二重サポート体制を整えています。このサポート方式により、お客様のトラブルに対して迅速かつ的確な対応が可能となります。主な特徴は以下の通りです：

- 専門チームによる24時間体制のサポート
- 技術的な問題に対する即時解析
- 多言語対応のサポートデスク

### リアルタイム稼働状況の確認
システムの現在の状態をリアルタイムで把握できるステータスページを提供しています。このページでは、以下の情報を確認できます：

- 各サービスの稼働状況
- 現在進行中の障害や メンテナンス情報
- 過去のインシデント履歴

トラブル発生時は、まずステータスページ（https://status.liveagent.com/）で全体の状況を確認することをおすすめします。システム全体に影響がない場合は、専用のサポートチャンネルまでご連絡ください。

お客様の迅速な問題解決をサポートするため、LiveAgentとFlowHuntの機能を活用し、効率的なサポート対応を実現しています。技術的な課題にも柔軟に対応し、最小限のダウンタイムでの解決を目指しています。
