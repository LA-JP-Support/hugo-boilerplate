## IPアドレス禁止機能の概要

### 機能説明
特定の訪問者IPアドレスをチャット、問い合わせフォーム、通話、提案ボード、フォーラム投稿からブロックし、不正行為や迷惑行為を防止する機能です。迷惑アクセスはボイスメールやスパムとしてリダイレクトされ、管理画面には禁止の履歴が保存されます。

## 主な機能と利点

### アクセスブロック
例えば、不正アクセスやスパム投稿を頻繁に行うIPを禁止リストに追加すれば、一切のやり取りが停止され、安全かつ効率的に対応が可能になります。

### 自動スパム対応
通話やチャットが自動的にスパム扱いされ、エージェントの手を煩わせません。

### 管理と追跡
管理画面には、禁止日時や理由、担当者、関連チケットIDなどの情報が記録されるため、対応の透明性も確保できます。

## 主要な利点

### マルチチャネル保護
- 複数チャネルにわたる標的IPのアクセスを一括でブロック

### 運用効率の向上
- スパム行為の自動処理によりエージェント負荷を軽減
- エージェントへの影響を最小限にし、安全性を強化

### 監査と管理
- 禁止履歴により監査や再評価も可能
- チケットIDと紐づけて対応履歴を管理
