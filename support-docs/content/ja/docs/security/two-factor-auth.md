はい、SmartWebのAIカスタマーサポートで使用するLiveAgentでは、Google Authenticatorを使った二要素認証（2FA）をご利用いただけます。

## 二要素認証のメリット

ログイン時にパスワードに加えて6桁のワンタイムコードを要求することで、アカウントの安全性を大幅に強化します。万が一パスワードが漏洩した場合でも、不正アクセスを防止できます。

## 利用可能な認証方法

- **Google Authenticator**&#8203;などの認証アプリと連携
- **GoogleやMicrosoftアカウントによるSSO連携**（シングルサインオン）により、間接的に2段階認証も適用可能

セキュリティリスクを軽減し、安心してカスタマーサポート業務を行えます。
