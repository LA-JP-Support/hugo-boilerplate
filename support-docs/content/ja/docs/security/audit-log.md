## 監査ログとは

システム内で行われたすべての操作を記録する機能です。誰が、いつ、どの操作をしたかを確認でき、セキュリティ管理や問題解決に役立ちます。

### 監査ログの主な特徴

たとえばチケットの削除や設定変更など重要なアクションもすべて記録されるため、トラブル発生時に責任の所在を特定できます。コンプライアンスや内部統制を強化する上でも重要です。

## 監査ログの重要なポイント

### 記録される情報

- すべての操作履歴を自動的に記録
- トラブル発生時に原因追跡が可能
- 不正利用やミスの防止に役立つ
- コンプライアンス強化に必須の機能
- 管理者が状況を把握しやすい
