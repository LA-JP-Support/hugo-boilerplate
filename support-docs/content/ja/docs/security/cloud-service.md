## サービス提供形態

SmartWebは**クラウドサービス（SaaS）のみ**&#8203;で提供しています。

| 提供形態 | 対応状況 |
|----------|----------|
| **クラウド版（SaaS）** | ✓ 提供中 |
| **オンプレミス版** | ✗ 未提供 |

## クラウド版のメリット

### すぐに始められる

| メリット | 説明 |
|----------|------|
| **即時利用開始** | サーバー構築不要、申込後すぐに利用可能 |
| **自動アップデート** | 常に最新機能を利用できる |
| **スケーラビリティ** | 利用量に応じて自動でスケール |

### 運用負荷の軽減

| メリット | 説明 |
|----------|------|
| **サーバー管理不要** | ハードウェア・OS管理が不要 |
| **保守作業不要** | パッチ適用・バックアップは自動 |
| **高可用性** | データセンターでの冗長構成 |

## データセンター

クラウド版でも、データの保管場所は選択可能です。

| リージョン | 場所 |
|-----------|------|
| **アジア** | シンガポール（日本のお客様推奨） |
| **US** | アメリカ |
| **EU** | ヨーロッパ |
