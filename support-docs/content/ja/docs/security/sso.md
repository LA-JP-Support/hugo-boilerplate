## シングルサインオン（SSO）とは

### SSOの基本概念
Google、Microsoft、Facebook、Oktaなどの認証プロバイダーを活用し、LiveAgentを含む複数システムへ一度のログインでアクセスできる認証機能です。セキュリティと利便性を高次元で両立します。

## SSOのメリット

### ログイン作業の簡略化
SSOを導入すると、複数パスワードの記憶や管理の手間がなくなり、ログイン作業が簡略化されます。ユーザーは一つの安全な認証で複数アプリケーションへのアクセスが可能になり、企業のセキュリティポリシーとも整合します。

### セキュリティ強化
また、二要素認証と併せることで、さらなる強固なセキュリティを実現できます。

## SSOの主な特徴

### サポート対象プロバイダー
- Microsoft
- Google
- Facebook
- Okta

### 主要な利点
- 複数システムを一つのIDで利用可能
- ログインの利便性とセキュリティを両立
- 多要素認証との連携で防御力も強化
- 管理コスト削減とユーザー体験向上に貢献
