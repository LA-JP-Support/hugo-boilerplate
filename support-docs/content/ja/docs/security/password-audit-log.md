## パスワード検証＆監査ログの概要

パスワードの強度を自動検証し、過去の漏洩・流出情報にも照らして安全性を確保できる機能。加えて、認証周りの操作を監査ログとして記録し、トラブル時の調査に活用可能です。

### 主な機能

ユーザーが弱いパスワードを設定しようとするとエラーや改善案が表示され、流出が疑われるパスワードの場合も警告が出されます。加えて、いつ誰がパスワード設定・変更を行ったのかがログに残るため、不正アクセス時の原因特定や再発防止に役立ちます。

## 主要な特徴

### パスワードセキュリティ強化

- パスワードの強度と流出履歴を自動チェック
- 弱いパスワードや流出パスワードの使用を防止

### 運用監査と透明性

- ログにより認証操作の透明性を確保
- セキュリティ事故時の迅速な原因追跡に貢献
- アカウント保護と運用監査の両方をサポート
