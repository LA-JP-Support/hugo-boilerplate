## 自動アップデート

SmartWebはクラウドサービス（SaaS）のため、**アップデートは自動実施**&#8203;されます。

| 項目 | 内容 |
|------|------|
| **バージョン管理** | 不要（自動） |
| **セキュリティパッチ** | 即時適用 |
| **新機能追加** | 自動反映 |
| **ダウンタイム** | 最小限 |

## 自動アップデートのメリット

| メリット | 説明 |
|----------|------|
| **常に最新** | 最新の機能とセキュリティを利用可能 |
| **運用負荷ゼロ** | サーバー管理・保守が不要 |
| **セキュリティ強化** | 脆弱性への即時対応 |
| **安定性向上** | パフォーマンス改善の自動実装 |

## アップデート情報の確認

アップデート内容は以下で確認できます。

| 情報源 | URL |
|--------|-----|
| **公式チェンジログ** | [changelog.liveagent.com](https://changelog.liveagent.com/) |

### 確認できる内容

- 新機能の追加
- 既存機能の改善
- セキュリティ強化
- バグ修正

## 計画メンテナンス

計画メンテナンスが必要な場合は、事前にステータスページで告知されます。

**ステータスページ**: [https://status.liveagent.com/](https://status.liveagent.com/)
