## IP アドレス制限の概要

SmartWebのエージェントパネルでは、セキュリティ強化のために IP アドレスによるログイン制限が可能です。この機能により、未承認のデバイスやネットワークからのアクセスを効果的に防ぐことができます。

### IP アドレス制限が可能かどうか

サポート担当者がエージェントパネルへログインする際の利用環境を IP アドレスで制限することができます。これにより、以下のようなメリットがあります：

- 不正アクセスのリスク低減
- 社内ネットワークからのみアクセス可能
- 特定の信頼できる場所からのログインを確保

### IP アドレス制限の詳細設定

IP アドレス制限は、企業のセキュリティポリシーに合わせて柔軟に設定できます。主な特徴は以下の通りです：

#### 柔軟な指定方法

ワイルドカード、アドレス範囲、サブネットマスクを利用した高度な設定が可能です。具体的には：

- 特定の IP アドレス（192.168.1.100）の許可
- IP アドレス範囲（192.168.1.0/24）の指定
- ワイルドカードを使用した部分一致（192.168.*.*)

例えば、本社オフィスのネットワーク（192.168.1.0/24）からのみアクセスを許可したり、在宅勤務者の特定の IP アドレスを追加したりすることができます。

設定は管理者パネルから簡単に行え、セキュリティと利便性のバランスを取ることができます。不正アクセスから組織を守りながら、正規のユーザーのアクセスを確保します。
