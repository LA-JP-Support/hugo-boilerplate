## 稼働率の目標

SmartWebは**99.9%以上の月間アップタイム**&#8203;を目標に運営しています。

| 項目 | 内容 |
|------|------|
| **目標稼働率** | 99.9%以上 |
| **計測期間** | 月間 |
| **実績** | 目標を一貫して達成 |

## 高稼働率を実現する仕組み

| 対策 | 説明 |
|------|------|
| **冗長構成** | データセンターでの複数サーバー構成 |
| **自動フェイルオーバー** | 障害発生時の自動切り替え |
| **24時間監視** | システム状態の常時監視 |
| **定期バックアップ** | データの安全な保管 |

## リアルタイムステータス確認

システムの稼働状況はリアルタイムで確認できます。

**ステータスページ**: [https://status.liveagent.com/](https://status.liveagent.com/)

### 確認できる情報

- 現在のシステム運用状態
- 各サービスコンポーネントの状況
- 過去の障害履歴と対応状況
- 計画メンテナンスの予定
