## 個人情報の取り扱いと保護

### LiveAgentのデータ管理

LiveAgentのデータは、GDPRに準拠し、暗号化されたデータベースで厳重に管理されます。お客様の個人情報は、最新のセキュリティ技術を用いて保護され、不正アクセスや漏洩から守られています。データは常に暗号化され、厳格なアクセス制御が行われています。

### GDPRの概要

#### GDPRとは

GDPR（General Data Protection Regulation: EU一般データ保護規則）は、EU（欧州連合）域内の個人データ保護と取り扱いに関する規則です。2018年5月25日に施行され、EU域内の個人データを扱うすべての組織に適用されます。GDPRにより個人が自身のデータに対する権利が強化され、企業はより厳格な個人データ保護が求められるようになりました。

主な特徴は以下の通りです：
- 個人データの透明性と同意
- データ主体の権利強化
- データ保護に関する厳格な規制
- 違反に対する高額な罰則

SmartWebでは、以下のような対策を講じています：
- データ暗号化
- 定期的なセキュリティ監査
- アクセス権限の厳密な管理
- プライバシーポリシーの明確な開示

個人情報の保護は、私たちにとって最も重要な責任の一つです。お客様の信頼を守るため、継続的にセキュリティ対策を強化しています。
