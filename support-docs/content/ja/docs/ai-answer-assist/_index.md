**AI回答アシスト（AI Answer Improver）**&#8203;は、オペレーターが作成した文章をAIが瞬時にブラッシュアップする機能です。チケット返信画面のマジックペンアイコンから利用できます。

## 主な機能

| 機能 | 説明 |
|------|------|
| **改善（Improve）** | 文章をより明確でプロフェッショナルに |
| **拡張（Extend）** | 詳細な説明や追加情報で充実 |
| **簡略化（Simplify）** | 複雑な内容を簡潔に要約 |
| **カスタム指示** | 独自の指示でAIに細かな調整を依頼 |

## フォーマリティ（文体）設定

- **カジュアル**: 親しみやすい表現
- **ニュートラル**: 中立的な表現（デフォルト）
- **ビジネスライク**: ビジネス向けの丁寧な表現

## AIプロバイダー

AI回答アシストは **FlowHunt** または **OpenAI** に対応しています。

| プロバイダー | 特徴 |
|-------------|------|
| FlowHunt | SmartWebプランに含まれる、AIメール回答作成との連携 |
| OpenAI | GPT-4等のモデルを選択可能、API利用料別途 |
