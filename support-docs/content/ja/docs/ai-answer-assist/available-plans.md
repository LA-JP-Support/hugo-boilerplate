## SmartWeb AIカスタマーサポートの一部として提供

AI回答アシスト（AI Answer Improver）は、**SmartWebのAIカスタマーサポート**&#8203;に含まれる機能の1つです。

SmartWebのAIカスタマーサポートは、以下の4つのAI機能をパッケージで提供しています。

| 機能 | 説明 |
|------|------|
| **AIチャットボット** | 24時間365日、顧客からの問い合わせに自動対応 |
| **AI連携チケットシステム** | メール・チャット・SNSからの問い合わせを一元管理 |
| **AIメール回答作成** | 顧客からのメールに対する返信下書きを自動生成 |
| **AI回答アシスト** | オペレーターの文章作成をAIが支援 |

## 料金体系

### 初期費用・月額費用

SmartWebのAIカスタマーサポートは、以下の料金体系で提供しています。

| 項目 | 費用 |
|------|------|
| **初期費用** | 20万円（税別）〜 |
| **月額費用** | 2万円（税別）〜 |

※具体的な費用は、お客さまのご要望をお伺いした上で、ご提案時に詳細をご提示いたします。

### 月額費用に含まれるもの

- LiveAgent・FlowHuntの利用料金
- AI回答アシスト（FlowHunt連携）の利用
- 設定変更・使い方サポート
- アカウント追加と設定

## AIプロバイダーと料金

AI回答アシストは、**FlowHunt**&#8203;または**OpenAI**&#8203;をAIプロバイダーとして選択できます。

| プロバイダー | 料金 | 備考 |
|-------------|------|------|
| **FlowHunt** | 月額費用に含まれる | SmartWeb標準、AIメール回答作成と連携可能 |
| **OpenAI** | 別途API利用料 | OpenAI社との直接契約が必要 |

### FlowHuntを利用する場合

SmartWebの月額費用に含まれているため、追加費用なしでご利用いただけます。

- AIメール回答作成（AI Answer Composer）との連携がスムーズ
- FlowHuntで学習したナレッジベースを活用可能
- 同一開発元（Quality Unit社）によるセキュリティ

### OpenAIを利用する場合

OpenAI社との直接契約が必要となり、API利用料金が別途発生します。

- OpenAI Platform でAPIキーを取得
- GPT-4、GPT-3.5-turbo等のモデルを選択可能
- 利用量に応じた従量課金

## 利用開始の流れ

AI回答アシストは、SmartWebのAIカスタマーサポートを導入いただくと、すぐにご利用いただけます。

1. **SmartWeb導入** - 初期設定をSmartWebチームが実施
2. **FlowHunt連携** - AIチャットボット・AIメール回答作成と同じナレッジベースを使用
3. **チケット画面で利用開始** - マジックペンアイコンからAI回答アシストを呼び出し
