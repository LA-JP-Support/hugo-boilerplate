## AIメール回答作成とAIチャットボットの違い

SmartWebでは、AIメール回答作成（AI Answer Composer）とAIチャットボットという2つのAI機能を提供しています。それぞれの特徴と使い分けを解説します。

## 比較表

| 項目 | AIメール回答作成 | AIチャットボット |
|------|-----------------|-----------------|
| 対応チャネル | メール・問い合わせフォーム | Webサイトのチャット |
| 出力方法 | 担当者向けに下書きを表示 | 顧客に直接回答 |
| 確認プロセス | 担当者が確認後に送信 | 自動で即時回答 |
| 最終責任 | 担当者 | AI（エスカレーション可） |
| 対応速度 | 担当者の確認後 | リアルタイム |
| 稼働時間 | 担当者の勤務時間 | 24時間365日 |

## AIメール回答作成の特徴

### 担当者主導の対応

AIが下書きを作成し、担当者が確認・編集してから送信します。最終的な回答内容は担当者がコントロールします。

### 適しているシーン

- 複雑な問い合わせ
- 正確性が特に重要な内容
- クレームや苦情対応
- 契約・法的な内容を含む問い合わせ
- 個別対応が必要なケース

## AIチャットボットの特徴

### 完全自動の対応

顧客の質問に対して、AIが24時間365日リアルタイムで回答します。担当者の介入なしで対応が完結します。

### 適しているシーン

- よくある質問（FAQ）
- 製品情報の問い合わせ
- 営業時間外の対応
- 多言語での対応
- 大量の問い合わせへの対応

## 使い分けのポイント

### 即時性が重要 → AIチャットボット

Webサイト訪問者からのリアルタイムな質問には、AIチャットボットが適しています。

### 正確性・慎重さが重要 → AIメール回答作成

内容を担当者が確認してから送信したい場合は、AIメール回答作成が適しています。

### 両方を組み合わせる

多くの企業では、両方のAI機能を併用しています。

**例:**
- Webサイトの問い合わせ → AIチャットボットが対応
- メールでの問い合わせ → AIメール回答作成が下書き、担当者が確認して送信
- AIチャットボットで解決しない場合 → エスカレーションしてAIメール回答作成で対応
