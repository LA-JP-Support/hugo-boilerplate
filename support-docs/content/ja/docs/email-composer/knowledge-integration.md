## ナレッジベースとの連携

AIメール回答作成（AI Answer Composer）は、AIチャットボットに事前学習させた製品情報、FAQ、対応マニュアルなどの**ナレッジベースをメール回答にも活用**&#8203;できます。

## 共有できるナレッジソース

FlowHuntのナレッジソース機能により、様々な情報源を設定・共有できます。

### Webサイトの定期クロール（Schedules）

指定したWebサイトを定期的にクロールし、最新情報を自動で学習します。

**活用例:**
- 製品情報ページ
- ヘルプセンター
- ブログ記事

### Q&A

よくある質問とその回答を登録し、AIが回答作成時に参照します。

**活用例:**
- カスタマーサポートFAQ
- 技術的な質問と回答
- 料金に関するQ&A

### ドキュメント

PDFやWordなどのドキュメントを学習させます。

**活用例:**
- 製品マニュアル
- 仕様書
- 社内ナレッジ

## メリット

### 情報の一元管理

一度学習させた情報がチャットボットとメール対応の両方で共有されるため、情報の二重管理が不要です。

### チャネル間の一貫性

チャネルを問わず一貫性のある高品質な顧客対応を効率的に実現できます。

### 運用負荷の軽減

ナレッジの更新も一箇所で完了するため、運用負荷を最小限に抑えられます。

## 連携の仕組み

```
[ナレッジベース]
    ↓ 学習
[FlowHunt]
    ↓ 連携
[AIチャットボット] ← 顧客（チャット）
    ↓ 同じナレッジを参照
[AIメール回答作成] ← 顧客（メール/フォーム）
```

## 注意事項

### ナレッジの品質

AIの回答品質は、学習させるナレッジの品質に大きく依存します。正確で最新の情報を維持するようにしてください。

### 更新の反映タイミング

Webサイトのクロールは、設定した頻度で実行されます。即座に反映したい変更がある場合は、手動でクロールを実行するか、Q&Aやドキュメントで直接更新してください。
