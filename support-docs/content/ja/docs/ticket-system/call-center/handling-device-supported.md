## LiveAgentのデバイスサポート概要

LiveAgentは、多様な通話機器をサポートする柔軟なコミュニケーションツールです。

### サポート対象デバイス
- デスクフォン
- ヘッドセット
- ソフトフォン

## メリット

### コスト最適化
企業は既存の機器をそのまま活用できるため、新規投資を最小限に抑えつつシステム導入ができます。柔軟な選択肢により、導入ハードルが大幅に下がります。

### 柔軟な運用
顧客対応も、利用シーンに応じて最適なデバイスを選べるため、安定した品質が保証されます。

## 主な特長
- デスクフォン、ヘッドセット、ソフトフォンなどに対応
- 既存機器を活用してコスト削減
- 環境に応じて最適な機器を選択可能
- 導入ハードルを下げ、移行を容易に
- 安定した通話品質を確保
