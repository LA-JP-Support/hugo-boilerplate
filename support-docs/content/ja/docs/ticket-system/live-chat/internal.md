## 内部チャットの概要
内部チャットは、エージェント同士がリアルタイムに連絡・相談できる社内向けチャットです。顧客対応中に非公開で確認事項をやり取りし、素早く合意形成できます。

ファイル添付やメンションで、担当外の専門家にも迅速に相談可能。対外返信の品質とスピードを両立します。

## 主な機能

### リアルタイムコミュニケーション
- エージェント間の非公開コミュニケーションを促進
- 顧客対応中でも裏側で確認・判断が可能

### 効率的な情報共有
- ファイル共有・メンションで迅速に連携
- 対応品質を維持しつつスピードを向上

## メリット

### 業務効率の向上
- 迅速な意思決定
- スムーズな情報交換

### ナレッジ管理
- 教育・ナレッジ共有の基盤としても有効
