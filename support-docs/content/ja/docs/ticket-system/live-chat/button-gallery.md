## チャットボタンギャラリーの概要
あらかじめ用意された多数のボタンテンプレートから選んで即時設置できます。デザイナーの工数をかけずに、ブランドに近い見た目を短時間で実装可能です。

### テンプレートの特徴
テンプレートは丸型・角型・吹き出し型などバリエーションが豊富で、開始文言やアイコンも差し替えられます。細部のカスタマイズと併用すれば、ガイドラインに沿った統一感のあるUIを素早く展開できます。

## 主な導入メリット

### 迅速な実装
- 用意済みテンプレートで短時間導入
- 導入初期のデザイン工数を削減

### カスタマイズ性
- 形状・サイズ・文言・アイコンを簡単に変更
- サイト全体で統一感のあるデザインを実現

### 運用の柔軟性
- 運用中の差し替え・最適化も容易
