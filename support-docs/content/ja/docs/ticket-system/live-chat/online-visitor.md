## オンライン訪問者の可視化

### リアルタイム訪問者モニタリング
現在サイトを閲覧している訪問者の数や、参照ページ、滞在時間などをリアルタイムに把握します。混雑状況や注目ページを可視化し、声がけの優先順位付けに活用できます。

### プロアクティブな顧客サポート
行動が活発な訪問者や離脱しそうな訪問者を見つけ、プロアクティブな支援を行うことで、チャット起点のコンバージョン向上を後押しします。

### 主な機能とメリット

#### リアルタイムの訪問者行動把握
- リアルタイムで訪問者数・行動を可視化
- 注目ページ・離脱兆候の検知に活用

#### サポート戦略の最適化
- プロアクティブな声がけの優先順位付け
- チャット起点のCV向上を支援
- サイト運用の改善指標としても有効
