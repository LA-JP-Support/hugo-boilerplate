## 「よく使うフレーズ」とは

「よく使うフレーズ」とは、事前に登録しておいた文章をキーワードを使って呼び出す機能です。メールやチャットの回答などで頻繁に使用するフレーズをテンプレートとして登録しておくと、同じフレーズや文章を何度も入力する必要がなくなるため、返答時間の短縮やミスタッチの防止と、対応効率が向上します。

## 「よく使うフレーズ」の使い方

### よく使うフレーズの呼び出し方法

- エージェントのチャット画面の文字入力フィールドにカーソルを合わせた状態で**「Ctrlキー ＋ スペースキー」**を押下すると、「よく使うフレーズ」のリストを呼び出すことができます。

　![](/liveagent/scripts/file.php?view=Y&file=3b5f13debf345ee09fb2ea04f9f872c8)

### フレーズの選択と検索

- リストからメッセージを選択すると、エージェント画面の文字入力フィールド内に反映されます。

- 「よく使うフレーズ」を検索するには、「よく使うフレーズ」のリストを表示した状態で、文字入力フィールドに2文字以上の検索ワードを入力してください。LiveAgent が適切なフレーズをサジェストします。

　　![](/liveagent/scripts/file.php?view=Y&file=bded6e2e3d55fa0a5a326b6ca978d403)

## 「よく使うフレーズ」の設定方法

### 設定手順

- エージェント画面左メニューより　設定＞自動化＞よく使うフレーズ　を選択します

- 「メッセージを作成する」をクリックして下記の画面で追加登録します

### メッセージの共有設定

- メッセージは「自分だけが利用」、「チームで利用」、「全体で利用」のいずれかを指定します
