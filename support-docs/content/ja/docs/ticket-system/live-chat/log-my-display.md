## チャットログ自動送信の無効化方法

### 初期設定の状態
顧客側のチャットウインドウには、デフォルトで「チャットログを自分宛に送る」というチェックボックスが表示されています。この設定は多くのユーザーにとって不要または煩わしい場合があるため、非表示または無効化したいというニーズがあります。

### チャットログ送信を無効にする手順

1. 「設定」に移動
2. 「電子メール」セクションを選択
3. 「顧客向けメールテンプレート」を開く
4. 「チャットの会話履歴」で「このメールを無効にする」をクリック

#### 無効化のメリット
チャットログの自動送信を無効にすることで、以下のようなメリットがあります：

- 不要なメールの削減
- プライバシーの保護
- メールボックスの整理

#### 注意点
チャットログの送信を無効にする際は、重要な対話記録を別の方法で保存する必要があるかもしれません。例えば：

- スクリーンショットの保存
- 社内共有システムへの記録
- 手動でのログ保存

#### おすすめの運用方法
チャットログの管理は、企業や利用シーンによって最適な方法が異なります。以下のポイントを考慮してください：

- 顧客サポートの性質
- 情報セキュリティポリシー
- チーム内のコミュニケーション方法

このように設定を調整することで、より効率的で快適なチャットサポート体験を実現できます。
