## 招待チャットとは

招待チャットとは、一定時間 Web ページに留まっているなどの条件を満たしたサイト訪問者に対して、チャットでの会話を促す画面を Web 上に表示させる機能です。招待画面はテンプレートをカスタマイズおよびオリジナル画像の利用が可能です。

※チャット機能は、[「プロフェッショナル」および「アドバンス」](http://liveagent.jp/plan.html)の2つのプランで利用できます。

## 招待チャットの設定方法

### チャット招待ボタンのテンプレート選択

エージェント画面左メニューより 「設定」＞「チャット」＞「招待」 を選択し、「作成」をクリックします。

サンプル画面で利用したいボタン形式を選びます。「＋ボタンを選ぶ」または「選択する」をクリックします。

ボタンには表示される位置によって異なるタイプが選べます。オリジナルの画像を使用したい場合はボタンスタイルの「カスタム」を選択します。

### チャット招待画面のカスタマイズ

サイト訪問者をチャットに招待する際に表示される画面です。画像やメッセージ、色などを指定します。

### チャットウィンドウのカスタマイズ

サイト訪問者が招待画面をクリックして、実際にチャットのやり取りを行なう画面です。タイトル、ウェルカムメッセージ、画面デザイン、表示位置、色などを指定します。

### 招待チャットの表示条件設定

まず、「ページの滞在時間」を5秒～30分の間で指定します。

次に以下の条件を and/or で組み合わせて設定します：
- URL
- リファラURL
- 訪問者が新規かリピーターか

### 招待チャットの概要設定

招待画面の名前、チャットを受信する割当先チーム、言語を選択します。「アクティブ」のチェックをオンにすることにより、招待チャット機能が動作します（チェックを外せば機能は停止します）。

※招待チャットが実装されると、このページに統計値（閲覧数、受入数、クリック率）が表示されます。

### ウェブサイトとの連携

「連携」を表示すると、ウェブサイトに連携するためのコードが表示されます。

表示されたコードをそのままウェブサイトのページへペーストしてください。ご利用のページの `</body>` の前にコードを配置します。

- ローカルファイルではボタンやページは表示できませんので、必ずウェブサーバにアップロードしてください。
