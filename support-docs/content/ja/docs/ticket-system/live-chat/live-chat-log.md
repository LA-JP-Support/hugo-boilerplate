## ライブチャットの会話ログ出力方法

### APIを使用したログ出力

SmartWebのライブチャットシステムでは、APIを通じて会話ログを簡単に出力できます。この機能により、カスタマーサポートチームは顧客との対話を効率的に管理し、詳細な分析や記録が可能になります。APIを使用することで、リアルタイムまたは過去の会話ログを柔軟に取得できます。

### サポートされているデータ形式

- JSON: JavaScript Object Notation形式で、軽量でデータ交換に最適
- XML: eXtensible Markup Language形式で、構造化されたデータ表現に適している

### ログ出力の具体的な手順

1. API認証キーの取得
   - SmartWebダッシュボードから専用のAPI認証キーを入手
   - セキュリティ設定で適切な権限を設定

2. APIリクエストの作成
   - エンドポイントURL: `/api/v1/chat-logs`
   - パラメータ：
     * 日付範囲
     * チャットエージェント
     * 顧客ID

3. データ形式の選択
   - JSONまたはXML形式を指定
   - レスポンスには以下の情報が含まれます：
     * メッセージ内容
     * 送信者情報
     * タイムスタンプ

### ログ出力のメリット

- 顧客サポート品質の向上
- トレーニングや品質管理に活用
- 法的要件や監査対応のための記録保持
