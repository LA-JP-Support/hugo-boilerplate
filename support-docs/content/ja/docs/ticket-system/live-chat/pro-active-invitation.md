## プロアクティブチャット招待の概要

プロアクティブチャット招待は、条件に合致した訪問者へ、こちらから先回りしてチャット招待を表示します。離脱前のフォローや高額商品の購入支援など、的確なタイミングでの介入が可能です。

招待の文言・タイミング・対象条件をチューニングすることで、不要なポップアップ感を抑えつつ効果を最大化できます。

## 主な特徴と利点

### 能動的な顧客アプローチ
- 条件ベースで能動的に声がけが可能
- 離脱防止や高単価商品の購入支援に有効
