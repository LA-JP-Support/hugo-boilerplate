## チャットウィンドウドッキングの概要

チャットウィンドウを画面端に固定して、閲覧中も常にアクセス可能にします。スクロールやページ遷移時の視認性を確保し、顧客が迷わず相談を継続できます。

モバイルでもUIの邪魔にならない配置を選べるため、ユーザビリティを損なわずにサポート導線を保てます。

## 主要な機能と利点

### アクセシビリティの向上
- 画面端に固定して常時アクセス可能
- スクロールやページ遷移でも視認性を維持

### デバイス最適化
- モバイル／デスクトップ双方で最適化
- UIの一貫性を保ち離脱を抑制

### サポート効果
- 相談継続率と解決率の向上に寄与
