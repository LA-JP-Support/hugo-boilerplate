## チャット招待ギャラリーの概要

条件に応じて自動表示できるチャット招待のテンプレート集です。滞在時間・離脱兆候・閲覧ページなどのトリガーで、自然に声がけできます。

テンプレートの文言や見た目を調整し、ページ目的に合わせた誘導を行うことで、放棄率の高い画面や高額商品ページでの支援を強化できます。

## 主な特長

### 柔軟な運用機能
- 目的別テンプレートを選んで即運用
- 滞在時間や動線に応じた発火条件を設定

### 最適化機能
- 文言のABテストで反応率を改善
- カゴ落ち・離脱を防止してCVR向上

### マルチパーパスな活用
- サポートとセールスの両面で活躍
