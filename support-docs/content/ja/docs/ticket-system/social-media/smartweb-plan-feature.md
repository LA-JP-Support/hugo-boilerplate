## SmartWebプランの概要

### プラン構成と機能

SmartWebの各プランでのSNS機能（Facebook、Instagram、X、WhatsApp、Viber）について説明します。

## プラン別詳細

### ベーシックプラン

- **SNS機能**: 非対応（追加オプションで対応可能）
- **標準機能**: チケットシステム、ライブチャット、ナレッジベース、メール統合

ベーシックプランは基本的なカスタマーサポート機能に特化しており、標準ではSNS連携機能は含まれていません。ただし、ビジネスニーズに合わせて追加料金でSNS機能を追加することが可能です。チケット管理やライブチャットを中心としたサポート体制を構築したい企業に最適です。

### コールセンタープラン

- **SNS機能**: 非対応（追加オプションで対応可能）
- **標準機能**: 電話・VoIP、IVR音声応答、通話録音、基本機能一式

コールセンタープランは電話対応を中心としたサポート体制の構築に最適化されています。VoIPシステムやIVR音声応答などの高度な通話機能が標準搭載されていますが、SNS連携は標準では含まれていません。電話サポートとSNS対応の両方を実現したい場合は、追加オプションでSNS機能を導入することが可能です。

### SNS/EC強化プラン

- **SNS機能**: 完全対応（Facebook、Instagram、X、WhatsApp、Viber）
- **標準機能**: 基本機能一式
- **特徴**: SNS連携に特化したプラン

SNS/EC強化プランは、ソーシャルメディア対応を最優先するビジネス向けに設計されています。Facebook、Instagram、X（旧Twitter）、WhatsApp、Viberなどの主要SNSプラットフォームとの完全統合を標準で提供し、すべてのSNSチャネルからの問い合わせを一元管理できます。デジタルマーケティングとカスタマーサポートの融合を目指す企業や、SNSを主要なコミュニケーションチャネルとする企業に最適なプランです。
なお、SNS機能でLINEをご利用頂く場合にはオプションとなります。

### AIチャットボット導入プラン

- **SNS機能**: 非対応（追加オプションで対応可能）
- **標準機能**: 24時間AI対応、基本機能一式

AIチャットボットプランは、24時間365日の自動応対を実現するAI技術に焦点を当てています。人的リソースを最適化しながら常時対応を可能にする先進的なプランですが、標準ではSNS連携機能は含まれていません。SNSからの問い合わせにもAIで対応したい場合は、追加オプションでSNS機能を導入可能です。
