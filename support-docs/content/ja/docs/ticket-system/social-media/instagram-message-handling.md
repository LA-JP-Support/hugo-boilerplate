## Instagramのダイレクトメッセージ対応について

### サポート状況の現状
申し訳ございませんが、Meta社のポリシーによりダイレクトメッセージングをサポートしていません。現在、SmartWebではInstagramのダイレクトメッセージ（DM）を直接連携することができません。これは、プラットフォーム間の技術的制約とMeta社のAPIポリシーによるものです。

### 代替的な連絡方法
お客様とのコミュニケーションを確保するため、以下の代替手段をご用意しています：
- メールサポート
- チャットボット
- お問い合わせフォーム
- 公式SNSアカウントでの対応

これらの方法により、迅速かつ効果的な顧客サポートを提供いたします。

### 今後の対応
SmartWebでは、今後のプラットフォーム拡張に向けて継続的に検討を進めています。Meta社のポリシー変更や技術的な進展に応じて、Instagramメッセージング機能の実装可能性を随時評価しています。

### お客様へのご案内
Instagram経由でのお問い合わせができない点、深くお詫び申し上げます。代替手段をご利用いただくことで、できる限り迅速かつ丁寧な対応を心がけております。お客様のご理解とご協力に感謝いたします。

お困りの際は、他の連絡手段をお選びいただくようお願いいたします。
