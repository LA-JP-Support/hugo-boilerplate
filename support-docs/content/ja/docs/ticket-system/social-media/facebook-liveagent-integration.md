## FacebookとLiveAgentの連携方法

### 概要
**FacebookとLiveAgentの連携設定手順についてご案内します。**

### 準備
#### LiveAgentへのログイン
**１．LiveAgentにログインします。**

### Facebookページの接続手順
#### Facebookページ設定画面への移動
**２．「設定」-「Facebook」-「Facebookページ」を開き、「Connect Facebook page」をクリックします。**

![接続画面](/liveagent/scripts/file.php?view=Y&file=a920d1979d45b7ac6a4f6f7955c93639)

#### Facebookログイン
**３．表示された画面でFacebookにログインします。（事前にFacebookにログインしていた場合、この画面は出ません）**

![Facebookログイン画面](/liveagent/scripts/file.php?view=Y&file=acdbfa897b348a3b10e444373f7ae191)

#### ページ接続の確定
**４．英語のメッセージ画面を閉じて、「Facebookページへ接続」のページで「接続」をクリックします。**

![接続確認画面](/liveagent/scripts/file.php?view=Y&file=1299c73bba1487e13dec8ff76323da2d)

### 連携後の設定
#### Facebookページの連携完了
**５．Facebookページが連携されました。**

![連携完了画面](/liveagent/scripts/file.php?view=Y&file=13ea2004aad544fe3dcc0bd43290bf42)

#### チーム設定
**６．上記画面の「編集」でチケットを割り当てる「チーム」の選択などが行なえます。**

![チーム設定画面](/liveagent/scripts/file.php?view=Y&file=83a1246ff062eacd3d7a3f547d5f1913)

### 追加情報
**※同様の手順で複数のFacebookページを登録し、連携することができます。**
