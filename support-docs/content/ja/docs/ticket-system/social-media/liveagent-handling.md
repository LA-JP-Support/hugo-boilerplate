## アクティブサポートとは

ソーシャルメディアのような、サポート窓口の外側で交わされる顧客の声に耳を傾け、問題解決をはかるサポート手法です。一般的なユーザーサポートの枠を超えて、顧客満足度や企業価値の向上につながる手法として、大企業を中心に認知されるようになりました。

LiveAgent は、現在多くあるソーシャルメディアのうち twitter と Facebook ページとの連携に対応しています。

### ソーシャルメディアサポートの課題

ソーシャルメディアのアカウントさえあれば気軽に始められるアクティブサポートですが、投稿の追跡や収集に意外と手間がかかるという難点があります。

例えば twitter のキーワード検索で、絶えず更新される投稿すべてを、サポート窓口の業務と並行しながら目を通すのは簡単なことではありません。また、応対のログについても、「返信」や「お気に入り」をタイムラインに流すだけでは、可視化された「顧客の声」を蓄積することもできません。

### LiveAgentによるソーシャルメディアサポートの解決策

LiveAgent は手間のかかる収集作業をチケットへ自動変換、サポート担当者がサポート応対に専念できる環境の構築を実現します。

## ソーシャルメディアプラットフォーム連携

### Twitter連携の特徴

サポート用アカウントへの通知（メンション）やキーワードの検索結果を、LiveAgent はリアルタイムですべてチケットへ変換・収集します。このチケットからエージェントは返信、リツイートなど顧客対応を行います。

チケットの管理は、例えば「タグ」と併用することで、顧客の声のポジティブ/ネガティブ度合をグラフ化したり、問合せ内容の傾向を統計にすることが可能です。また、タイムラインへの新規投稿から顧客とコミュニケーションを図ることもできます。

【関連項目】　twitter と連携してツイートを取り込むには, タイムラインへの新規投稿

### Facebook Page連携の特徴

企業・団体の facebook ページに投稿されるコメントなどを、すべてチケットへ変換・収集します。このチケットへ返信したり「いいね」をすることで顧客対応を行います。

LiveAgent との連携は管理者または所有者権限のエージェントが既存の facebook ページを登録設定するのみで、一般エージェント個人のfacebookアカウントをfacebookページと連携する必要はありません。
