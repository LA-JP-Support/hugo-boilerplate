## SNS広告との連携概要

はい、SNS広告との連携が可能です。

### サポートしている広告プラットフォーム

- Facebook/Instagram広告投稿へのコメント
- 広告経由のメッセンジャー問い合わせ

## 連携機能の主な特長

### 自動チケット管理
- **自動チケット化**: 広告への反応を自動でチケット化
- **迅速な対応**: 広告コメントへの返信を一元管理
- **効果測定**: 広告別の問い合わせ件数を把握

## 活用シナリオ

### 具体的な運用メリット
- キャンペーン広告への問い合わせを専任チームで対応
- 広告のコメント炎上を早期発見・対応
- 広告効果のKPIとして問い合わせ数を計測

## ご利用について

SNS連携機能はオプションとなります。詳しくはお問い合わせください。
