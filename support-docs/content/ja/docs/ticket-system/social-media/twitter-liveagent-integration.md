## Twitter と LiveAgent の連携手順

### はじめに
**TwitterとLiveAgentの連携設定手順についてご案内します。**

### 連携手順

#### LiveAgentにログイン
**１．LiveAgentにログインします。**

#### Twitterアカウントの追加
**２．「設定」-「Twitter」-「Twitterアカウント」を開き、「Twitterアカウントを追加」をクリックします。**

![](/liveagent/scripts/file.php?view=Y&file=4c5dc71168c314d06ea672a0d52be835)

#### Twitterアカウントの認証
**３．表示された画面でTwitterアカウントにログインし、「連携アプリを認証」をクリックします。**

![](/liveagent/scripts/file.php?view=Y&file=787a8d2127b8e65cb1b7fe12b25f45a8)

#### 連携完了
**４．Twitterアカウントが連携されました。**

![](/liveagent/scripts/file.php?view=Y&file=e803bee7fcb25efbab8b510606767aa4)

#### アカウント設定の編集
**５．上記画面の「編集」でアカウント連携の有効化やチケットを割り当てる「チーム」の選択などを行います。**

![](/liveagent/scripts/file.php?view=Y&file=6ec14706492132a605a6bbe7bd0ca8f9)

### 補足
**※同様の手順で複数のTwitterアカウントを登録し、連携することができます。**
