## Viber統合の概要

Viber統合は、世界中で利用されるメッセージングアプリ「Viber」とLiveAgentを接続し、メッセージのやり取りを一元管理できる機能です。顧客は日常的に使うアプリから直接問い合わせができ、企業はそれを通常のサポートフローに組み込めます。

## Viberの特長と利点

### グローバル市場でのコミュニケーション
Viberは特にヨーロッパやアジア圏で人気が高く、国際的に事業展開する企業にとって有効なチャネルです。LiveAgentで統合することで、異なる地域の顧客との円滑なコミュニケーションが実現します。

### 効率的なサポートワークフロー
加えて、既存のチケットシステムと同じ流れで処理できるため、担当者はツールを切り替える必要がなく、効率が向上します。

## 主な特徴

### メッセージ管理
- ViberメッセージをLiveAgentで一元管理

### 顧客利便性
- 顧客は普段使うアプリから気軽に問い合わせ可能

### グローバル対応
- 国際市場での顧客対応に最適
- 地域ごとの顧客ニーズに柔軟対応

### 業務効率
- チケットフローに統合され効率化
