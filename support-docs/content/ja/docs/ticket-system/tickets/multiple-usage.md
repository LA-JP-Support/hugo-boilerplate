## 複数オペレーターの同時利用

はい、LiveAgentは**複数のオペレーターが同時に利用できる**&#8203;チーム向けのヘルプデスクシステムです。

## Agent Collision Detection（エージェント衝突検出）

LiveAgentには**Agent Collision Detection**&#8203;機能が搭載されており、複数のオペレーターが同じチケットに対応しようとした場合の混乱を防ぎます。

### 表示される情報

| 状態 | 表示内容 |
|------|----------|
| **閲覧中** | 誰がそのチケットを閲覧しているか |
| **回答中** | 誰がそのチケットに回答を作成中か |

### 動作の仕組み

1. **リアルタイム検出** - チケットを開くと、他のオペレーターにその状況が即座に通知
2. **重複回答の防止** - すでに対応中のチケットを開こうとすると警告が表示
3. **To Solveアルゴリズム連携** - 自動振り分けと連携し、対応中のチケットは他のオペレーターに割り当てられない

## チーム利用のメリット

| メリット | 説明 |
|----------|------|
| **効率向上** | 同じチケットへの重複作業を排除 |
| **顧客体験向上** | 1つの問い合わせに複数回答が届く混乱を防止 |
| **チーム連携** | 誰が何に対応中かリアルタイムで把握 |

## 利用可能なプラン

Agent Collision Detection機能は、LiveAgentの**全プラン**（Small、Medium、Large、Enterprise）でご利用いただけます。

## アカウント追加について

チームメンバーを増やす場合は、オペレーターアカウントの追加が必要です。

| 項目 | 内容 |
|------|------|
| **最小アカウント数** | 3名〜 |
| **追加方法** | 管理画面から随時追加可能 |
| **料金** | アカウント数に応じた月額費用 |

SmartWebのプランでは、LiveAgentのアカウント追加も含めてサポートいたします。

## 関連機能

### デパートメント（部門）管理

チームを部門ごとに分けて、問い合わせを自動振り分けできます。

- 営業部門、サポート部門、技術部門など
- 部門ごとの営業時間設定
- 部門間でのチケット転送

### 権限管理

オペレーターごとに異なる権限を設定できます。

- 管理者：全機能へのアクセス
- エージェント：チケット対応のみ
- カスタムロール：必要な権限のみ付与
