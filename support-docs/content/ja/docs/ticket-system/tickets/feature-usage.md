# GeoIP機能を利用する

## GeoIPとは

GeoIP機能を使うと、サイト訪問者の位置情報を自動で取得・表示できます。

## 取得できる情報

### 位置情報の詳細
- 訪問者の国
- 大まかな都市名
- 使用言語の推定

## 活用メリット

### ビジネス活用シーン
- 海外からの問い合わせを英語対応チームに自動振り分け
- 地域別の問い合わせ傾向を分析
- 時差を考慮した対応時間の把握

## 設定方法

管理画面の「設定」→「システム」→「GeoIP」から有効化できます。

## 利用する際の注意点

### 精度に関する制限
- IPアドレスベースの判定のため、100%正確ではありません
- VPN利用者は実際と異なる位置が表示される場合があります
