## LiveAgentとGmailの接続方法

### クラウド版LiveAgentでのGmail接続

#### 接続手順
クラウド版のLiveAgentアカウントは、GoogleのOAuth認証を使用してGmailアカウントに接続します。ポップアップウィンドウでこの接続を認証するとGmailメールボックスが接続され、これ以上設定する必要はありません。

1. LiveAgentにログインし、メールアカウントの設定画面を開きます。
2. 「設定」＞ 「電子メール」＞「メールアカウント」を選択
3. 「メールアカウントの追加」をクリックし、Gmailのコネクタを選択
4. ポップアップウィンドウでGmailアカウントを承認
5. メールボックスの「チーム」を指定して保存

### ソフトウェア版LiveAgentでのGmail接続

#### 接続の基本手順
LiveAgent（ソフトウェア版）では、Gmailのユーザー名とパスワードを使用してIMAP / POP3とSMTPサーバを経由して接続します。

1. LiveAgentにログインし、メールアカウントの設定画面を開く
2. 「メールアカウントの追加」をクリックし、Gmailコネクタを選択
3. Gmailのメールアドレスとパスワードを入力
4. 接続ボタンをクリック

### Gmail接続の要件と設定

#### IMAP接続の有効化
1. ブラウザでGmailにログイン
2. 「設定」＞「メール転送とPOP/IMAP」セクションを開く
3. IMAPアクセスを有効にする

#### セキュリティ設定の調整
- Googleアカウントのセキュリティ通知を確認
- Google アカウントへのアクセスを許可
- 必要に応じて、安全性の低いアプリへのアクセスを設定

### パスワード変更時の対応

#### 接続の再設定
Gmailアカウントのパスワードを変更した場合:
1. 「設定」＞「電子メール」＞「メールアカウント」を開く
2. 該当のメールアカウントを編集
3. 新しい認証情報で接続を更新

### トラブルシューティング
接続に問題が続く場合は、サポートに連絡してください。
