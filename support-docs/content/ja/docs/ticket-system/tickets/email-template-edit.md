## メールテンプレートの概要

電子メールからの回答を迅速に行うために、 LiveAgent ではメールテンプレートを作成し、システム内にストックすることができます。メールテンプレートには、顧客向けとエージェント向けの二種類があります。

## メールテンプレートの設定

### メールテンプレート画面への移動
エージェント画面左メニューより、設定＞電子メール＞顧客向けテンプレートを選択します

### 顧客向けメールテンプレートの編集
- 編集したいメールテンプレートを選び、編集をクリックします
- チーム固有のテンプレートを作成する場合は、「＋チーム固有のテンプレートを作成する」ボタンをクリックします
- メール設定のオプション：
  - チケットの解決をメールで知らせる
  - チャットの会話ログの取得

### メール本文の編集
- メールテンプレートの書式選択（プレーンテキストまたはHTML）
- 変数の利用
  - `{$firstname}` のような変数を使用
  - 変数によってチケットの条件や対応エージェントに応じたデータを差し込み可能

### メールテンプレート編集時の注意点
- 編集前に元のテンプレートをバックアップ
- 「引用符オプション」の活用
- 「テスト送信」機能の利用

## メールテンプレートの変数一覧

### 利用可能な差し込み項目
| データ種類 | 変数 |
|------------|------|
| サーバ情報 | サーバの日付、サーバの時間 |
| チケット情報 | チケットの件名、チケットコード、チケットID、チケットの状態、チケットのURL |
| 受取人情報 | 受取人氏名（姓/名）、受取人メールアドレス |
| エージェント情報 | エージェントID、エージェントの姓/名 |
| その他 | メッセージ、依頼者からの最終メッセージ |

### 受取人氏名の変数に関する注意
- 氏名は半角スペースで区切ることで名と姓に分割可能
- 変数の順序に注意（最初の部分が名、後の部分が姓）

## エージェント向けテンプレートの編集
- エージェント向けテンプレートの編集手順は顧客向けテンプレートと同様
- 編集したいテンプレートを選択し、「編集」をクリック
