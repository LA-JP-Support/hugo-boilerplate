## 休止機能（Pause）の概要

一時的にチケットの自動割り当てを停止する機能です。休憩や他業務に集中する際に活用できます。

エージェントが対応できない状態で新しいチケットが割り当てられるのを防ぐため、業務の混乱を回避できます。短時間の離席でも安心して利用可能です。

## 主な機能と利点

### 自動割り当ての一時停止
- 自動割り当てを一時停止可能
- エージェントの業務状況に応じて柔軟に対応

### 業務効率の向上
- 休憩や他業務に集中できる
- 業務の混乱を防止
- エージェントの負担を軽減

### 働き方のサポート
- 柔軟な働き方をサポート
- 短時間の離席にも対応可能
