## スマートチケットのタグ機能とは

任意のタグをチケットに追加することで、問合せの多い製品の特定や問合せ内容の分類を省力化できます。

## タグの管理と使用方法

### タグの作成手順

エージェント画面左メニューより　設定＞自動化＞タグ＞作成　を選択します。

- 「作成」ボタンをクリックし、「タグの名前」、「文字色」、「背景色」を設定すると作成完了です。名称変更など編集時は、各タグの「編集」をクリックして行います。

### チケットへのタグ追加

作成したタグを受信したチケットに挿入するには、チケットを開き「+」をクリックし適切なタグを選択し「ウィンドウを閉じる」をクリックします。チケット画面を確認すると、タグが追加されています。

### タグ付けの自動化

ルーティン作業となるタグ付けは、ルールの設定により自動化することも可能です。
