## スパム判定されたチケットの復元方法

スパム判定されたチケットは、エージェント画面から手動でチケット一覧へ復元することができます。

【関連項目】　スパムの自動検出について

### チケット復元の手順

1. エージェント画面での検索

- エージェント画面左メニューより 「検索」＞「チケット」 を選択後、「カスタム検索条件」をクリックします

![](/liveagent/scripts/file.php?view=Y&file=f24f4301a19ae08970cdefc258248520)

### スパムチケットの絞り込み

- 検索条件の「ステータス」で、「スパム」のみにチェックを入れたのち、「適用する」をクリックします

### チケットの復元

- スパム除外されたチケットが一覧表示されます

- 復元したいチケットを選択後、「スパムではない」をクリックすることで、チケットを一覧画面へ復元することができます
