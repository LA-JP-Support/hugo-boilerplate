## エージェント追加の概要

トライアル版ではログイン後のエージェント画面内から、エージェント（ログイン ID）の追加を行うことができます。

## エージェント追加の手順

### エージェント作成画面へのアクセス

- 左メニューから 「設定」＞「エージェント」＞「エージェントを作成」 をクリックします。

### エージェント情報の入力

- エージェントの名前、メールアドレスを入力し、役割と性別を選択します（写真の追加は任意）。

### エージェント作成の完了

- 「作成」をクリックすると、入力したアドレス宛に通知のメールが送られます。受信したメールでパスワードを設定すれば追加設定は完了します。
