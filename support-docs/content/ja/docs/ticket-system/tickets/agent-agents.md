## エージェントとは

LiveAgentにおける「エージェント」は、顧客からの問い合わせに対応する担当者を指します。

### エージェントの役割
メール、チャット、電話、SNSなど複数のチャネルを通じて届く問い合わせを処理する役割を担います。

## エージェント管理の特徴

### フレキシブルな追加と削除
たとえば小規模チームで利用を開始し、業務拡大に伴って新しいエージェントを次々と招待することができます。無制限で追加可能なため、スタートアップから大規模企業まで柔軟に対応できます。

### エージェント管理の主要ポイント

#### チャネル対応
- メール、チャット、電話、SNSすべてのチャネルで対応可能

#### スケーラビリティ
- 無制限にエージェントを追加できる柔軟性（エージェント単位で費用がかかります）

#### 権限管理
- 管理者(Admin)やエージェントなど権限を分けられる

#### 個別管理機能
- 通知設定、所属部門、プロフィール編集など個別管理が可能

#### チーム拡張
- チーム拡張にあわせて迅速に体制を強化できる
