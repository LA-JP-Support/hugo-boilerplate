## スマートチケット分割の概要

スマートチケットを分割することで、複雑な問合せや異なる進捗状況のチケットを効率的に管理できます。

### 分割が推奨されるシーン

1つのチケット上にある複数の問合せについて、次のような状況で分割が有効です：

- 複数の問合せを個別のチケットとして管理する
- 問合せごとに異なる進捗ステータスがある
- 一部の問合せを他のチームやエージェントへ転送（エスカレーション）する必要がある

## チケット分割の手順

### チケット分割の基本操作

- 分割したいチケットの詳細画面を表示します。右上の「▼」から「分割」を選択すると、これまでの応対ログを継承した、新たな管理番号のチケットが作成されます。

分割後の 2つのチケットは引き続き関連付けられており、リンクを辿って参照することができます。

### エスカレーションの実施

- 分割されたチケットの詳細画面より「転送」をクリックして、エスカレーション先のチームやエージェントを指定します。分割時にチームやエージェントを割り当てることで、スムーズなエスカレーションが可能となります。
