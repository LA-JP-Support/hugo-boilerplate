はい、SmartWebのAI連携チケットシステムでは、複数のチケットを一つに統合（マージ）できます。

## チケット統合が役立つ場面

- 同じ顧客がメールとチャットで同じ問題を問い合わせてきた場合
- 同一内容について複数のチケットが作成された場合
- 問い合わせが分散して管理が煩雑になった場合

## 統合のメリット

- **重複対応の防止**：複数の担当者が同じ問い合わせに対応するミスを防げます
- **履歴の一元化**：すべてのやり取りを一つのスレッドで確認可能
- **対応品質の向上**：顧客に同じ説明を繰り返させることがありません

## 操作方法

1. 統合したいチケットを開き、「その他」ボタンから「Merge」を選択
2. 統合先のチケット番号を入力
3. 必要に応じてタグや受信者の統合オプションを選択
4. 「Merge」ボタンをクリックして実行

**注意**：統合は元に戻せません。実行前に統合先のチケット番号をご確認ください。
