## 対応デバイスとOS

LiveAgent は、インターネット接続環境があれば Windows、Mac のようなデスクトップ/ノート PC のほか、iPhone/iPad/Android などのスマートフォンやタブレット端末にも対応しています。

### デスクトップ環境

#### Windows
- Microsoft Windows 10以降

#### Mac
- Apple Mac OS X

### モバイル環境

#### スマートフォン・タブレット

- Apple iOS
- Google Android

## 利用時の機能制限

### モバイルブラウザの制限

- スマートフォン、タブレットPCからブラウザーでの利用ではチャットの受信はできません。チャットをご利用になりたい場合には下記の専用アプリをご使用ください。

- モバイルビューでは、チケットの閲覧と処理だけが利用できます。他の機能を利用される場合には、デスクトップビューに切り替える必要があります。

- 通話機能はご利用いただけません。

## モバイルアプリ

LiveAgent は iOS・Android向けの無料専用アプリをご用意しています。アプリ版ではチャットの受信・応答が可能になります。

### アプリダウンロード

- [iOS](https://itunes.apple.com/jp/app/live-agent/id661142245?mt=8)
- [Android](https://play.google.com/store/apps/details?id=com.qualityunit.liveagent)

### アプリの主な機能

- チケットの閲覧
- チケットの解決、転送、返信
- チャット
- チケットの選択機能(チーム)
- タグ、ノート、チームの閲覧
