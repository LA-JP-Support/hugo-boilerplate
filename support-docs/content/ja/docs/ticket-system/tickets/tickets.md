## チケット内GIFとは

チケット内でアニメーションGIFを利用できる機能です。静的な説明に比べ、動きのあるビジュアルで状況をわかりやすく伝えられます。

### GIFの利点

例えば不具合の再現手順をGIFで共有すれば、顧客もエージェントも状況を簡単に理解できます。テキストだけでは伝わりにくい問題も視覚的に補足でき、解決までの時間を短縮できます。

### 主な特徴

**ポイント：**

- チケット内にGIFアニメーションを挿入可能
- 不具合や操作手順を直感的に共有できる
- テキストだけでは伝わりにくい情報を補足
- 問題解決までの時間を短縮
- 顧客体験を向上させる効果がある
