## ノート機能の概要

チケットに内部用のメモを追加できる機能です。顧客には表示されず、担当者間の引き継ぎや注意事項の共有に役立ちます。

### 使用例

例えば「この顧客は過去に返金対応を受けている」といった補足情報を残すことで、次の担当者も背景を理解した上で対応できます。

## 主な機能

### 非公開メモの追加
- 内部用の非公開メモを追加可能

### コミュニケーション支援
- 引き継ぎや注意事項の共有に有効
- チーム全体で一貫性ある対応を実現

### 情報管理
- 顧客に余計な情報を見せずに管理可能
- 対応履歴の精度を向上
