## 統合受信ボックス（Universal inbox）の概要

### 機能の定義
メール、チャット、電話、SNSなどすべてのチャネルからの問い合わせを一元的に集約して管理する機能です。エージェントは複数ツールを切り替える必要がなくなります。

### 利用シーンの例
例えば顧客が午前にメールで質問し、午後にSNSで追加連絡をしても、統合受信ボックスではすべて1つの画面に表示されます。やり取りを見落とさずにスムーズに対応できます。

## 主な利点

### チャネル管理の効率化
- 複数チャネルの問い合わせを一元管理
- 複数ツールを切り替える必要なし

### カスタマーサポートの改善
- 顧客対応の効率と正確性を向上
- 見落としや重複を防止
- 一貫した対応で顧客満足度を改善
