## ファイル添付の概要

LiveAgentでは、複数の形式の添付ファイルを簡単に送受信できます。ビジネスコミュニケーションにおいて、添付ファイル機能は重要な役割を果たし、スムーズな情報共有を可能にします。

## ファイル添付の方法

### 添付手順
LiveAgentからメール送信時は、送信フォーム下部の「クリップ」のアイコンをクリックして、ファイルの添付を行います。以下の手順で簡単に添付できます：

- アイコンをクリック
- ローカルデバイスからファイルを選択
- 添付するファイルを確認

### 添付ファイルの保存
添付されたファイルは、送信・受信に関わらず、関連するチケット内に自動的に保存されます。これにより、以下のメリットがあります：

- 履歴管理が容易
- 過去のファイルに簡単にアクセス可能
- コミュニケーションの追跡が簡単

## 添付ファイルの制限

### ファイルサイズ
添付可能なファイルの容量は、1回の送受信につき最大 128 MB となります。大容量のファイル共有にも対応しており、以下のようなファイルタイプに対応しています：

- 画像ファイル（JPG、PNG、GIF）
- ドキュメント（PDF、DOCX、XLSX）
- 圧縮ファイル（ZIP、RAR）

このファイル添付機能により、LiveAgentユーザーは効率的で柔軟なコミュニケーションを実現できます。重要な情報や資料をスピーディに共有することが可能です。
