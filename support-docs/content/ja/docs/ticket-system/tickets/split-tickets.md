## チケット分割（Split tickets）の概要

チケット分割は、一つのチケットを複数に分割して、それぞれ異なる担当者や部門に割り当てられる機能です。複数の課題が含まれている場合に役立ちます。

### 使用例

例えば顧客が「ログイン不具合」と「請求に関する質問」を同時に送信した場合、それぞれを独立したチケットに分割して専門部署に割り当てられます。

## メリット

### 課題の柔軟な対応
- 複数課題を独立したチケットに分割可能
- 専門性の高い部門に正しく振り分け可能

### 業務効率の向上
- 対応スピードを向上
- 顧客への回答精度を高める
- 複雑な問い合わせを整理できる
