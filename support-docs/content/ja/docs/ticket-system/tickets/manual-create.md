## スマートチケットの手動作成について

LiveAgent のチケットは、各設定済みチャネルから作成されるチケットのほか、エージェント自身が新規作成をすることもできます。新規作成が可能なチケットの種類は次の通りです。

### 新チケット

「新チケット」から作成されるチケットは、顧客へ電子メールを新規作成するのと同義です。「宛先（＝顧客）」「送信者（チームでの設定可）」「件名」と「本文」を作成する必要があります。

### ナレッジベース関連の作成

#### 新しいナレッジベースの記事

ナレッジベースへ投稿する記事を新規作成できます。作成記事は LiveAgent のカスタマーポータル（FAQ サイト）または内部向けナレッジベースのどちらかを選択して掲載します。エージェント画面の「ナレッジベース」より「記事」を作成するのと同等です。

#### フォーラムのトピックを作成

ユーザーフォーラムへ掲載するトピックの作成を行います。この機能は、エージェント画面の「ナレッジベース」より「フォーラム」を作成するのと同等です。

#### 提案を作成

フィードバック（提案）カテゴリーを新規作成します。この機能は、エージェント画面の「ナレッジベース」より「フィードバックカテゴリー」を作成するのと同等です。

### 内部チケットの作成

#### 新しい内部チケット

エージェントやチーム間で共有するチケットを作成します。作成には「チーム」「エージェント」「件名」「ノート」および「顧客」を入力します。

「顧客」欄の入力は任意ですが、入力するとこのチケットは顧客情報に関連付けられます。新規作成時は内部メモ（ノート）扱いのため、顧客情報を関連付けても顧客へメールを自動送信することはありません。必要があれば「返信」の操作から顧客へメール送信が可能です。

### ソーシャルメディア投稿

#### 新規ツイート

LiveAgent に Twitter アカウントを連携している場合に表示されます。投稿したい Twitter アカウントをプルダウンメニューから選択してメッセージを送信すると、指定アカウントのタイムライン上に投稿が反映されます。

#### Facebook に投稿

LiveAgent に Facebook アカウントを連携している場合に表示されます。投稿したい Facebook ページをプルダウンメニューから選択してメッセージを送信すると、指定の Facebook ページのタイムライン上に投稿が反映されます。※URL リンク添付時のサムネイル画像やページ情報の表示は未対応です。

### 通話機能

#### 新規 Twilio 通話

LiveAgent に Twilio を連携している場合に表示されます。「宛先（顧客の電話番号）」を入力し、「発信者」番号を選択すると、LiveAgent から電話をかけることができます。
