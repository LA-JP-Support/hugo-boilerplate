## キーボード ショートカットの概要

LiveAgentでは、生産性を向上させる様々なキーボードショートカットが利用可能です。以下に、主要なショートカットを機能別に分類しています。

### ナビゲーション ショートカット

- **CTRL + H** - ホームタブに移動する
- **CTRL + ↑** - 開いている次のチケットタブに移動
- **CTRL + ↓** - 開いている前のチケットタブに移動

### チケット管理 ショートカット

- **CTRL + D** - 「解決すべきチケット」を開く（解決すべきチケットがある場合）
- **CTRL + E** - チケットを解決する
- **CTRL + Q** - チケットウインドウを閉じる
- **CTRL + 左クリック** - バックグラウンドでチケットを開く（チケット一覧画面からのみ）

### メッセージ操作 ショートカット

- **CTRL + I** - 「メール定型文」のリストを表示する（新規チケットを作成またはチケットに返信時）
- **CTRL + K** - 「よく使うフレーズ」のリストを表示する（チケットに返信またはチャット対応時）
- **CTRL + SPACE** - 「よく使うフレーズ」のリストを表示する（CTRL + Kと同じ）
- **CTRL + S** - メッセージを送信する

### チャット ショートカット

- **CTRL + M** - チャットに応答する
- **CTRL + SHIFT + M** - チャットにバックグラウンドで応答する
