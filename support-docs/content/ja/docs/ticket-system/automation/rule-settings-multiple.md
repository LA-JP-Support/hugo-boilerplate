## 複数キーワード登録の方法

### 正規表現を活用したキーワード設定

ルールの適用条件を作成する際、複数のキーワードを1つの設定項目に登録するには、正規表現の機能が便利です。正規表現は、文字列のパターンマッチングを高度に制御できる強力なツールで、SmartWebのルール設定において柔軟な条件指定を可能にします。

具体的には、キーワードを「|（パイプ）」で区切ることで、複数のキーワードを同時に検索できます。例えば、「apple|banana|orange」と設定すれば、これら3つのキーワードのいずれかにマッチするコンテンツを抽出できます。

#### 正規表現の使用例
- キーワード: `セール|キャンペーン|プロモーション`
- マッチする文字列: 「夏のセール開催中」「キャンペーン実施中」「プロモーション情報」
