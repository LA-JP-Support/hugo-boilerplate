## 「コンタクトグループ」とは

「コンタクトグループ」は、カスタマーサポートチームが顧客情報を効率的に管理し、整理するための強力な分類ツールです。チケットに関連付けられた「コンタクト」や「企業」に対して、視覚的かつ論理的なタグ付けを可能にする機能です。このグループ分けにより、エージェントは顧客の特性や重要度をすぐに把握できます。

## コンタクトグループの作成方法

### 設定画面への移動

エージェント画面左メニューより、「設定」＞「自動化」＞「コンタクトグループ」を選択します。この画面で、新しいグループの作成や既存グループの管理を行うことができます。

### グループの新規作成と編集

「作成」ボタンをクリックし、以下の詳細を設定します：

1. グループの名前：わかりやすく、目的を反映した名称を付けます
   - 例：「VIP顧客」「新規顧客」「要注意案件」など
2. 文字色：グループ名の表示色を選択
3. 背景色：グループを識別しやすい色を設定

具体的な活用シーン：
- 顧客の優先度に応じたグループ分け
- 特定の属性や特徴による分類
- サポート対応の効率化とチーム内コミュニケーション改善

グループの編集や名称変更は、各グループの「編集」ボタンをクリックして行えます。この柔軟な管理機能により、常に最新の顧客情報分類が可能です。
