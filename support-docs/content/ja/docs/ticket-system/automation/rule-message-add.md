## メッセージグループの種類について

### メッセージグループの種類とは

「設定」＞「自動化」＞「ルール」の作成画面において、「適用」で「メッセージが追加された」を選択すると、「メッセージグループの種類」という条件があります。※表示される項目は[ご契約プラン](https://www.liveagent.jp/pricing/)により異なります。（下図はアドバンスプランの例）

## メッセージグループの種類の詳細

### チャット関連

#### チャット
- チャットを開始した

### チケット管理

#### 削除
- チケットを削除した

#### 内部チケット
- 内部チケットを作成した

#### オフライン関連
- オフライン内部：（オフラインチケットの通知を受け取ったメーラーからエージェントが返信したときに）依頼者には返信が届かず内部通知としてメッセージが追加された
  - この条件は「メール設定」で「チケットの受信者へは返信を送信しません」にしている場合に動作します

![](/liveagent/scripts/file.php?view=Y&file=d705a19d07bdd2c1355e67383869bed4)

- オフライン：チャットのオフラインメッセージからチケットが作成された

#### チケットステータス
- 解決する：チケットを解決した
- スパム：エージェントまたはシステムがチケットをスパムとしてマークした

### チケット作成関連

#### チケット初期情報
- スタート情報：チャットボタン/フォーム/通話ボタン経由の新規チケットが作成された、またはカスタマーポータルにログインしたユーザから提出された新規チケットが作成された
  - メール、電話、エージェントによる新規チケット作成、新規内部チケット、新規twilio通話、フォーラムトピック作成、新規ツイートは除く

### チケット転送

#### 転送関連
- 転送（Transfer）：チケットを他のエージェント（またはチーム）に割り当てた
- 転送（Forward）：チケットからメールを転送した
- 転送先へ返信：チケットのメール転送先から返信があった

### その他の操作

- 分割：チケットを分割した
- 延期：チケットを延長した

### ソーシャルメディア・外部サービス関連

- Facebook：Facebookからチケットが作成された
- ナレッジベース：フォーラムトピックまたはフィードバックから作成されたチケットが更新された
- ナレッジベースの開始：フォーラムトピックまたはフィードバックから新規チケットが作成された

### 通話関連

- 電話（受電）：電話（受電）からチケットが作成された
- 電話　内部：日本では該当機能がありません
- 電話　発信：LiveAgentから発信した電話でチケットが作成された

### その他

- タグ：チケットにタグが追加された
- Twitter：Twitterからチケットが作成された
- リツートする：チケットからリツートした
