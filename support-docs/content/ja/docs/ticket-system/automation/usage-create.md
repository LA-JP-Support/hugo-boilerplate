## 「よく使うフレーズ」とは

「よく使うフレーズ」は、電子メールへの返信やライブチャット、内部チケットなど LiveAgent 上でメッセージを作成時に呼び出すことのできるテンプレートです。類似機能に「メール定型文」がありますが、こちらはHTML タグ編集やファイル添付といった機能が利用できる点で異なります。

「よく使うフレーズ」は、簡単な挨拶や質問などエージェントの言葉遣いを共通化するのに適しています。

## 「よく使うフレーズ」の使い方

### よく使うフレーズの呼び出し方

「よく使うフレーズ」を利用するには、エージェント画面のメッセージ返信フォーム右下の「吹き出し」アイコンをクリックします。

〔右図の赤丸に囲まれた「吹き出し」アイコンをクリックすると、雛形メッセージを呼び出すことができます〕

![](/liveagent/scripts/file.php?view=Y&file=603fa41ce70b7d592cd2ea7d162f0137)

### よく使うフレーズの検索方法

呼び出された「よく使うフレーズ」はリストとしてポップアップ表示されます。該当メッセージをクリックすると、エージェント画面の返信フォーム内に反映されます。目当てのフレーズがすぐに見つからない場合は、リスト上部の「入力して検索」へ冒頭の文字列を入力してください。

例えば、「他に何かございますか？」を検索する際は「他に...」とメッセージの初めの文字列を入力します。

![](/liveagent/scripts/file.php?view=Y&file=5162471281c0b70f439b2939a8a19559)

## 「よく使うフレーズ」の作成方法

### よく使うフレーズ一覧画面へのアクセス

エージェント画面左メニューより 「設定」＞「自動化」＞「よく使うフレーズ」 を選択すると、下図のような雛形メッセージ一覧画面が表示されます。作成済みメッセージの内容を編集する場合は、各メッセージをクリックします。

![](/liveagent/scripts/file.php?view=Y&file=b342e431778766568c12fd0bc33cc2e3)

### 新規フレーズの作成

「メッセージを作成する」をクリックして、メッセージの新規作成を行います。「メッセージ」の入力と、メッセージの「利用可能」範囲（全ユーザー、チーム １つまたは自分）を選択してください。検索用「キーワード」は任意で設定します。

![](/liveagent/scripts/file.php?view=Y&file=086942fb3a10c2eb46d06aceb9d766b0)
