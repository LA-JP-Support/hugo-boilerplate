## タイムルールとは

タイムルール（Time rules）は、一定間隔で自動的に動作するルールを設定できる機能です。条件を満たしたチケットに対し、定期的に処理を実行します。

### 主な特徴と使用例

例えば「1時間ごとに未対応チケットを確認してアラートを送信」といった自動化が可能です。これにより抜け漏れを防止できます。

### 主なメリット

**ポイント：**

- 定期的に実行される自動ルールを設定可能
- 未対応チケットへの自動アラート送信が可能
- 抜け漏れを防ぎ対応品質を確保
- エージェントの負担を軽減
- SLA遵守を支援
