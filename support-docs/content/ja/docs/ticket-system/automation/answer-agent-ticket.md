## チケット自動割り当てルールの概要

エージェントがチケットに対応（回答）すると自動的にそのエージェントに割り当てるルールの設定をご案内します。

![](/liveagent/scripts/file.php?view=Y&file=4b51deb6c2009430be06e52bb31769c2)

## チケット自動割り当てルールの重要性

### 手動割り当ての課題
新規チケットに回答する際、エージェントがチケット内の ***「その他」＞「自分に割り当てる」 ***の手動操作をし忘れると、そのチケットは特定のエージェントに割り当てられていない状態で保存されます。

### ルール適用のメリット
このルールを作成することで、以下のメリットがあります：
- チケットの割り当て漏れの防止
- オープンチケットの効率的な管理

## ルールの動作詳細

![](/liveagent/scripts/file.php?view=Y&file=b478c44467acc7139b38fb82efeeb43c)

### チケット割り当ての自動化
このルールが適用されると、エージェントがチケットに対応した後に自動で対応者にチケットが割り当てられます。

### 既存割り当ての上書き
オープンチケットで既に別のエージェントに割り当てられている場合は、このルールを適用すると最後に対応したエージェントに割り当てを変更します。
