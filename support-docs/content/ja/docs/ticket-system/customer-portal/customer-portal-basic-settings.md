## カスタマーポータルの基本設定

カスタマーポータルページのデザインのカスタマイズ、アクセス解析用の JavaScript コード追加、問い合わせボタンやチャットボタン設置などの設定が行なえます。

### サイト設定を開始する

**「カスタマーポータル」‐「設定」‐「デザインをカスタマイズ」**

![](/liveagent/scripts/file.php?view=Y&file=5b30559598d18ee9975fe1f91e3f7d02)

### ブランド設定

ロゴとタイトルを登録します。

![](/liveagent/scripts/file.php?view=Y&file=bf1fd5a2ca5d396da6bc44f7efd15494)

### デザインカスタマイズ

ベースとなるテーマを3パターンから選択

このセクションでは他に色のカスタマイズ、独自のヘッダ／フッタやCSSコードを挿入できます。

![](/liveagent/scripts/file.php?view=Y&file=f52da7d2e8ec576138db30cacb9e240e)

### トラッキングコード設定

利用したいスクリプトを入力します。たとえばカスタマーポータル上にチャットボタンや問い合わせフォームボタンを表示したい場合は、ボタンの「連携」に表示されたHTMLコードをコピー＆ペーストします。

![](/liveagent/scripts/file.php?view=Y&file=a7a3e84dd05683fb61d850d0378398d1)
