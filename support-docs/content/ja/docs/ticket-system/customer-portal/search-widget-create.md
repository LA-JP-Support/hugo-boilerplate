## 検索ウィジェットの概要

カスタマーポータルの記事コンテンツを検索するウィジェット（検索窓）の作成方法について説明します。このウィジェットにより、顧客は問合せメッセージの代わりにキーワード検索で迅速に目的の回答にたどり着くことができます。

## 検索ウィジェットの作成手順

### ステップ1：検索ウィジェットの新規作成
「カスタマーポータル」＞「検索ウィジェット」で「作成」ボタンをクリックします。

![](/liveagent/scripts/file.php?view=Y&file=465025f1fe85260c9913e422e09ac5c3)

### ステップ2：基本設定
「概要」セクションで以下の項目を設定します：
- ナレッジベースのカテゴリー選択
- 言語指定
- ウィジェット名の入力
- オプション設定
  - 「記事を新しいウインドウで開く」
  - 「アクティブ」チェックボックス

![](/liveagent/scripts/file.php?view=Y&file=e1cf2cdc148346cf0e5056a5a470d93b)

### ステップ3：デザインのカスタマイズ
「デザイン」セクションで以下の項目をカスタマイズできます：
- タイトル
- 検索ボックスのヒントテキスト
- 色設定
- カスタムCSS

![](/liveagent/scripts/file.php?view=Y&file=cbddf8013f3205e9f852dec0ab4ee143)

### ステップ4：ウェブページへの統合
提供されたコードをウェブページの </body> タグ直前に挿入します。

![](/liveagent/scripts/file.php?view=Y&file=a5f71c2b7942eeb752ae10116134964d)
