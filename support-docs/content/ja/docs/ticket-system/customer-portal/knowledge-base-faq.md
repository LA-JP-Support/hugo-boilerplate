## ナレッジベース - FAQ（Knowledge Base）の概要

ナレッジベースは、顧客向けに情報提供プラットフォームを構築するための柔軟で強力な機能です。

### 主な機能と特徴

顧客向けに記事、フォーラム、フィードバックボードなどをまとめたカスタマイズ可能な情報提供プラットフォームを構築する機能です。テーマやデザインの設定も柔軟に可能です。

### コンテンツ管理の柔軟性

たとえば、FAQ記事をカテゴリー分けして整理し、必要に応じて記事に画像・添付やスタイルを設定することができます。エージェント向けには非公開の内部記事も併設でき、情報共有の幅が広がります。

### メリットと効果

顧客サポートの「顔」として機能する高品質なナレッジ拠点を短時間で構築できます。

## 主要な特徴

**ポイント：**

- FAQ記事・フォーラム・フィードバックを統合
- デザインテーマもカスタマイズ可能
- 公開／非公開コンテンツを柔軟に管理
- 顧客／エージェント向け情報を一括管理
- 自己解決促進により問い合わせ削減へ貢献
