SmartWebのAIチャットボットとAIメール回答作成（AI Answer Composer）は、**FlowHuntのナレッジソース**&#8203;から情報を取得し、RAG（検索拡張生成）技術を使って正確な回答を生成します。

## ナレッジソースの種類

FlowHuntでは3つの主要なナレッジソースを管理できます：

| ナレッジソース | 説明 | 用途 |
|---------------|------|------|
| **Schedules（スケジュール）** | Webサイトの定期クロール | 製品情報、FAQ、ブログなど |
| **Q&A（質問と回答）** | 手動で登録する質問と回答 | よくある質問、重要な回答 |
| **Documents（ドキュメント）** | ファイルアップロード | マニュアル、PDF、Excel等 |

## 対応コンテンツ形式

### ファイルアップロード対応形式

| 形式 | 説明 |
|------|------|
| PDF | PDF文書 |
| DOCX / DOC | Word文書 |
| XLS / XLSX | Excel表計算 |
| TXT | テキストファイル |
| RTF | リッチテキスト |
| CSV | カンマ区切りデータ |
| Markdown | マークダウン形式 |
| HTML | Webページ形式 |
| YouTube動画 | 動画の文字起こし |

### Webクロールの種類

| タイプ | 説明 |
|--------|------|
| ドメインクロール | 指定ドメイン全体を自動巡回 |
| 単一URLクロール | 特定の1ページのみ取得 |
| サイトマップクロール | sitemap.xmlを基に巡回 |
| YouTubeチャンネル | チャンネル内の動画を取得 |
| URLリストクロール | 指定したURL一覧を取得 |
| XML商品フィード | ECサイトの商品データ取得 |

## Webクロールの設定

### 更新頻度の選択

| 更新頻度 | おすすめの用途 |
|----------|---------------|
| 毎日（Daily） | ニュースサイト、頻繁に更新されるFAQ |
| 毎週（Weekly） | 製品情報、ブログ |
| 毎月（Monthly） | マニュアル、会社情報 |
| 毎年（Yearly） | 変更頻度の低い情報 |

※変更がない場合でも半分のクレジットが消費されます

### 詳細設定オプション

| オプション | 説明 |
|-----------|------|
| ブラウザ使用 | JavaScript描画が必要なページ用（追加クレジット） |
| リンク追跡 | ページ内のリンク先も取得 |
| スクリーンショット取得 | ページの画像を保存（追加クレジット） |
| プロキシローテーション | IPを変えてクロール（追加クレジット） |
| 除外URL | 指定パターンのURLをスキップ |
| 対象URL限定 | 指定パターンのURLのみ取得 |
| カスタムヘッダー | 認証等のHTTPヘッダーを追加 |

## Q&A管理システム

### Q&A運用の特徴

- **即座の反映**: 手動で追加したQ&Aは即座に利用可能
- **優先度設定**: よくある質問を優先的に提示
- **バージョン管理**: 回答の変更履歴を管理
- **承認フロー**: 重要な情報は承認後に反映

## より良い回答のためのコンテンツ作成ガイドライン

AIの回答精度は、学習させるコンテンツの品質に大きく依存します。以下のガイドラインに従ってコンテンツを整備することで、回答の精度が向上します。

### 基本的な文章作成ルール

- **1文は短く**（15〜20語目安）
- **専門用語は避ける**
  - 例：「計画の実施を促進」→「計画を進める手助けをする」
- **曖昧な表現を避ける**（「など」「〜の場合がある」を減らす）
- **能動態を使用する**（受動態は避ける）

### コンテンツ構成のコツ

- 見出しで内容を区切る
- 箇条書きを活用する
- よくある質問をすべてカバーする
- 適度な余白を入れる

### 導入前の準備チェックリスト

- [ ] FAQやマニュアルの内容が最新か確認
- [ ] 専門用語を平易な言葉に置き換え
- [ ] 見出し・箇条書き・短い段落で構成を最適化
- [ ] よくある質問パターンの網羅性をチェック
- [ ] 曖昧な表現や不十分な説明を修正

**ヒント**: 既存のFAQやマニュアルは、導入前に一度見直すことをおすすめします。古い情報や曖昧な表現がないか確認しましょう。

## ナレッジベース共有のメリット

AIチャットボットとAIメール回答作成は**同じナレッジベースを共有**&#8203;しています：

- **一度の整備で両方に対応**: FAQ・マニュアルを登録すれば、チャットでもメールでも活用
- **回答の一貫性**: どのチャネルでも同じ品質の回答を提供
- **メンテナンスの効率化**: ナレッジの更新が両方に即座に反映
