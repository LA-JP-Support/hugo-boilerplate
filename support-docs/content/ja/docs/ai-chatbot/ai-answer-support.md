## AI回答アシスト（AI Answer Improver）の概要

AI回答アシスト（AI Answer Improver）は、**オペレーター（担当者）の文章作成を支援する機能**です。顧客に直接対応するAIチャットボットとは異なり、担当者が作成した回答文をAIが瞬時にブラッシュアップします。

SmartWebの全てのプランでご利用いただけます。

## AI回答アシストの主な特徴

### 1. 4つの文章調整オプション

担当者が作成した文章を、以下の4つのモードで最適化できます：

| モード | 説明 |
|--------|------|
| **改善（Improve）** | 文章をより明確でプロフェッショナルに磨き上げ |
| **拡張（Extend）** | 詳細な説明や追加情報で文章を充実させる |
| **簡略化（Simplify）** | 複雑な内容を簡潔で分かりやすく要約 |
| **カスタム指示** | 独自の指示でAIに細かな調整を依頼 |

### 2. 3段階のフォーマリティ選択

文章のトーン（文体）を状況に合わせて調整できます：

- **カジュアル**: 親しみやすい表現
- **ニュートラル**: 中立的な表現
- **ビジネスライク**: ビジネス向けの丁寧な表現

### 3. 多言語翻訳サポート

作成した回答を他の言語に翻訳できます。外国語でのメール対応や、多言語サポートが必要な場合に便利です。

### 4. カスタムプロンプト機能

「もっと丁寧に」「専門用語に解説を追加」「英語に翻訳して」といった独自の指示を入力して、AIに文章を調整させることができます。

## 活用シーン

### シーン1: 複数資料から洗練された回答を作成

**課題**: 複数のマニュアルやFAQから情報を集めると、文体やトーンがバラバラになる

**AI回答アシストによる解決**: 
- 集めた情報を入力
- 「改善」モードで文章を統一
- 読みやすく整理された一貫性のある返信文を作成

### シーン2: 専門用語のわかりやすい説明

**課題**: 技術的な内容を、専門知識のない顧客にわかりやすく説明したい

**AI回答アシストによる解決**:
- 技術的な回答を入力
- カスタム指示で「専門用語に解説を追加」と指定
- 初心者向けの表現に変換された回答を取得

### シーン3: 外国語メール対応

**課題**: 英語や中国語での問い合わせに対応する必要があるが、外国語が苦手

**AI回答アシストによる解決**:
- 日本語で回答を作成
- カスタム指示で「英語に翻訳して」と入力
- 自然な表現の外国語回答を作成

## AIメール回答作成（AI Answer Composer）との連携

AI回答アシスト（AI Answer Improver）は、AIメール回答作成（AI Answer Composer）とシームレスに連携します。

1. **AIメール回答作成**が会話履歴とナレッジベースから回答案を自動生成
2. 生成された回答案がそのまま**AI回答アシスト**の編集画面に引き継がれる
3. ゼロから文章を書く必要がなく、AIが生成した下書きをブラッシュアップするだけ

2つのAI機能の組み合わせで、対応時間の大幅短縮と品質向上を同時に実現します。

## AIチャットボットとの違い

| 項目 | AI回答アシスト | AIチャットボット |
|------|---------------|-----------------|
| 対象 | オペレーター（担当者） | 顧客（エンドユーザー） |
| 動作 | 手動で使用 | 24時間自動稼働 |
| 目的 | 文章作成支援 | 自動応答 |
| 出力 | オペレーターが確認・編集してから送信 | 顧客に直接回答 |

## 利用上の注意

- AI回答アシストの出力は、必ずオペレーターが確認してから顧客に送信してください
- 機密情報や個人情報を含む文章の取り扱いには注意が必要です
- OpenAI連携の場合、OpenAI社との直接契約が必要となり、API利用料金が別途発生します
