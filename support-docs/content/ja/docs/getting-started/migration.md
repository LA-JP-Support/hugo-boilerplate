SmartWebのAIカスタマーサポート機能（AIチャットボット、AIメール作成、AI回答アシストなど）を利用するには、チケットシステムをLiveAgentに移行する必要があります。

## 移行プラグインで簡単に切り替え

ZendeskやFreshdeskなど主要なチケットシステムからの移行プラグインが用意されています。プラグインを利用すれば、既存のFAQデータやチケット履歴もそのままインポート可能です。

## 移行の流れ

1. **現行システムからデータをエクスポート** - 既存のチケット履歴、FAQデータ、顧客情報などを出力
2. **LiveAgentへインポート** - 移行プラグインを使って自動的にデータを取り込み
3. **AIカスタマーサポート機能の設定** - FlowHuntとの連携でAI機能を有効化
