## モバイルアプリの概要

LiveAgentのモバイルアプリで、**外出先でもチケット対応・チャット返信**&#8203;が可能です。

| 対応OS | デバイス | ダウンロード |
|--------|----------|--------------|
| **iOS** | iPhone / iPad | App Store |
| **Android** | スマートフォン / タブレット | Google Play |

## 主な機能

### チケット管理

| 機能 | 説明 |
|------|------|
| **チケット対応** | 新規チケットの確認・返信・解決 |
| **チケット転送** | 他のエージェントや部門への転送 |
| **フィルター** | プリセットフィルターでチケット整理 |
| **タグ・メモ** | タグやエージェントメモの確認 |

### リアルタイム通知

| 機能 | 説明 |
|------|------|
| **プッシュ通知** | 新規問い合わせを即時把握 |
| **チャット通知** | 顧客からのメッセージをリアルタイム受信 |
| **ステータス更新** | チケットの状態変化を通知 |

### チャット対応

| 機能 | 説明 |
|------|------|
| **ライブチャット** | 顧客とのリアルタイムチャット |
| **チャット返信** | 外出先でも即座に返信可能 |
| **チャット履歴** | 過去のやり取りを確認 |

## モバイルアプリのメリット

| メリット | 説明 |
|----------|------|
| **場所を選ばない** | オフィス外でも顧客対応が可能 |
| **即時対応** | プッシュ通知で迅速に対応開始 |
| **業務継続性** | 移動中や外出先でもサポート品質を維持 |
| **機会損失防止** | 対応の遅れによる顧客離脱を防止 |

## 利用シーン

- **外出中の緊急対応** - 急な問い合わせにも即座に対応
- **リモートワーク** - 自宅やカフェからサポート業務
- **営業訪問中** - 移動時間を有効活用
- **イベント会場** - 現地での顧客サポート

## ダウンロード方法

### iOS（iPhone / iPad）

App Storeで「LiveAgent」を検索してダウンロード

### Android

Google Playで「LiveAgent」を検索してダウンロード

## 初期設定

1. アプリをダウンロード・インストール
2. LiveAgentアカウントでログイン
3. プッシュ通知を許可
4. 利用開始
