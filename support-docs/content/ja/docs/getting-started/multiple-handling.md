## 複数の支社・店舗でのサポート対応

はい、複数の支社・店舗での利用に対応しています。

### デパートメント（部門）管理

店舗や支社ごとに「デパートメント」を作成し、問い合わせを自動で振り分けできます。

### 活用シナリオ

#### 問い合わせ管理の柔軟性
- 東京本社、大阪支社、福岡支社でそれぞれ独立した問い合わせ管理
- 店舗Aと店舗Bで別々のチャットボタンを設置
- 部門ごとの営業時間・休日設定

### レポート分析

#### 詳細な performance 可視化
- 支社・店舗ごとの問い合わせ件数
- 対応時間の比較分析
- 全社横断でのレポートも可能

### 導入コスト

複数部門の設定は追加料金なしで利用可能です。
