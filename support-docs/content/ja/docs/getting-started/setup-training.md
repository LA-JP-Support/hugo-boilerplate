## トレーニングの概要
はい、導入時にトレーニングを実施しています。

## トレーニング対象者別の内容

### 管理者向けトレーニング
- システム設定
- ユーザー管理
- レポートの見方

### オペレーター向けトレーニング
- 基本操作
- チケット対応
- チャット応対の方法

### AIチャットボット トレーニング
- 学習データの登録・更新方法

## トレーニング実施形式

### 研修方法
- オンラインでのリモート研修（Zoom等）

### ドキュメント提供
- 操作マニュアルの提供
- 動画マニュアルも順次整備中

## サポート体制

### 導入後のサポート
- 不明点はサポート窓口で対応
- 追加トレーニングも対応可能（別途費用）

## カスタマイズオプション
お客様の業務に合わせたカスタマイズトレーニングも承ります。
