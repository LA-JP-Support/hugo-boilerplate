## SmartWebの最適な対象企業

### 企業規模と利用形態
中小企業から部署単位での利用まで幅広く適しています。

### ターゲットとなる企業の特徴
特に「顧客対応を強化したいが専門のIT人材が少ない」企業や、「SNSからの問い合わせが増えて手が回らない」企業に最適です。

### 対応可能な業種
ECサイト、小売、ITサービス、製造業など業種を問わず活用できます。
