## 3層構造のサポート体制

SmartWebは**3層構造の充実したサポート体制**&#8203;を提供しています。

| 層 | 担当 | 対応言語 | 内容 |
|----|------|----------|------|
| **1層** | インターワーク（SmartWeb） | 日本語 | 一次窓口・総合サポート |
| **2層** | LiveAgent社 | 英語（24時間） | 技術的な高度な問題 |
| **3層** | FlowHunt社 | 英語 | AI機能の専門サポート |

## 日本語サポート（インターワーク）

**お客様は基本的に日本語サポートのみで完結**&#8203;できます。

### 対応内容

| 内容 | 説明 |
|------|------|
| **一次窓口** | 日本人担当者が対応 |
| **代行対応** | LiveAgent社・FlowHunt社への問い合わせを代行 |
| **導入後チューニング** | 機能アップのご相談も無料 |
| **継続改善** | ビジネス成長に合わせた最適化提案 |

### お客様のメリット

- 言語の壁を感じることなく安心して利用
- 海外ベンダーとのやり取りを代行
- 日本のビジネス習慣に合わせた対応

## LiveAgent社のサポート

技術的な高度な問題が発生した場合、LiveAgent社の専門チームが対応します。

| 項目 | 内容 |
|------|------|
| **対応時間** | 24時間体制 |
| **対応言語** | 英語 |
| **問い合わせ方法** | インターワーク経由で対応可能 |

## FlowHunt社のサポート

AIチャットボット機能に関する専門的なサポートを提供します。

| 項目 | 内容 |
|------|------|
| **対象** | AIチャットボットプラン利用者 |
| **対応言語** | 英語 |
| **問い合わせ方法** | インターワーク経由で対応可能 |

## 継続的なサポート

導入後も継続的なサポートを提供します。

| サポート内容 | 説明 |
|--------------|------|
| **改善提案** | 利用状況に応じた最適化提案 |
| **機能アップ相談** | 新機能の活用方法 |
| **追加トレーニング** | 必要に応じて実施（別途費用） |
