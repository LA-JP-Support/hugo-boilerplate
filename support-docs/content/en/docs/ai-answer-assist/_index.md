**AI Answer Assist (AI Answer Improver)** is a feature that allows AI to instantly refine text created by operators. It can be accessed from the magic pen icon in the ticket reply screen.

## Main Features

| Function | Description |
|------|------|
| **Improve** | Make text clearer and more professional |
| **Extend** | Enrich with detailed explanations and additional information |
| **Simplify** | Summarize complex content concisely |
| **Custom Instructions** | Request detailed adjustments from AI with custom instructions |

## Formality (Writing Style) Settings

- **Casual**: Friendly expressions
- **Neutral**: Neutral expressions (default)
- **Business-like**: Polite expressions for business use

## AI Providers

AI Answer Assist supports **FlowHunt** or **OpenAI**.

| Provider | Features |
|-------------|------|
| FlowHunt | Included in SmartWeb plan, integrates with AI email response creation |
| OpenAI | Selectable models like GPT-4, separate API usage fees |
