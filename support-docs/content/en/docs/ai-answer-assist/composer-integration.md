## Integration of Two AI Features

AI Answer Assist (AI Answer Improver) seamlessly integrates with AI Email Response Composer (AI Answer Composer) to achieve more efficient response creation.

## Integration Flow

### Step 1: Automatic Response Draft Generation

AI Email Response Composer (AI Answer Composer) automatically generates response drafts from ticket conversation history and knowledge base.

### Step 2: Transfer to Editing Screen

The generated response draft is directly transferred to the AI Answer Assist (AI Answer Improver) editing screen.

### Step 3: Polish and Refine

Use AI Answer Assist features to further optimize the response draft.

You can instantly adjust with instructions such as:

- "Make it more polite"
- "Make it more concise"
- "Add explanations for technical terms"

### Step 4: Send

When satisfied, click the "Use for Response" button to insert into the ticket reply, then send after final confirmation.

## Benefits of Integration

### No Need to Write from Scratch

Since you only need to polish the AI-generated draft, the burden of writing is significantly reduced.

### Dramatic Reduction in Response Time

With a consistent workflow from automatic response draft generation to text optimization, response time can be dramatically shortened.

### Quality Improvement

AI Email Response Composer retrieves accurate information from the knowledge base, while AI Answer Assist refines the text, enabling efficient creation of high-precision responses that empathize with customers.

## Differences from Standalone Use

| Usage Method | Benefits | Suitable Scenarios |
|--------------|----------|-------------------|
| AI Answer Assist Only | Quickly improve existing text | When staff has already created a draft |
| Integrated Use | Automate response creation from scratch | New inquiry responses, when maximum efficiency is desired |
