## Provided as Part of SmartWeb AI Customer Support

AI Answer Assist (AI Answer Improver) is one of the features included in **SmartWeb's AI Customer Support**.

SmartWeb's AI Customer Support provides a package of the following four AI features:

| Feature | Description |
|---------|-------------|
| **AI Chatbot** | 24/7/365 automated response to customer inquiries |
| **AI-Integrated Ticket System** | Centralized management of inquiries from email, chat, and social media |
| **AI Email Response Creation** | Automatic generation of draft replies to customer emails |
| **AI Answer Assist** | AI assistance for operator text creation |

## Pricing Structure

### Initial and Monthly Costs

SmartWeb's AI Customer Support is offered with the following pricing structure:

| Item | Cost |
|------|------|
| **Initial Cost** | From 200,000 yen (excluding tax) |
| **Monthly Cost** | From 20,000 yen (excluding tax) |

### What's Included in Monthly Cost

- LiveAgent and FlowHunt usage fees
- AI Answer Assist (FlowHunt integration) usage
- Configuration changes and usage support
- Account addition and setup

## AI Providers and Pricing

AI Answer Assist allows you to choose between **FlowHunt** or **OpenAI** as the AI provider.

| Provider | Pricing | Notes |
|----------|---------|-------|
| **FlowHunt** | Included in monthly cost | SmartWeb standard, can integrate with AI Email Response Creation |
| **OpenAI** | Separate API usage fees | Direct contract with OpenAI required |

### Using FlowHunt

No additional costs as it's included in SmartWeb's monthly fee.

- Smooth integration with AI Email Response Creation (AI Answer Composer)
- Can utilize knowledge base learned in FlowHunt
- Security by the same developer (Quality Unit)

### Using OpenAI

Direct contract with OpenAI is required, with separate API usage fees.

- Obtain API key from OpenAI Platform
- Choose models like GPT-4, GPT-3.5-turbo
- Pay-as-you-go pricing based on usage

## Getting Started

AI Answer Assist is available immediately upon implementing SmartWeb's AI Customer Support.

1. **SmartWeb Implementation** - Initial setup performed by SmartWeb team
2. **FlowHunt Integration** - Uses the same knowledge base as AI Chatbot and AI Email Response Creation
3. **Start Using in Ticket Interface** - Call AI Answer Assist from the magic pen icon
