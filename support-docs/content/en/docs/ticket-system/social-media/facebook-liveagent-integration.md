## How to Integrate Facebook with LiveAgent

### Overview
**We will guide you through the setup process for integrating Facebook with LiveAgent.**

### Preparation
#### Logging into LiveAgent
**1. Log into LiveAgent.**

### Facebook Page Connection Procedure
#### Navigating to Facebook Page Settings
**2. Go to "Settings" - "Facebook" - "Facebook Pages" and click "Connect Facebook page".**

![Connection screen](/liveagent/scripts/file.php?view=Y&file=a920d1979d45b7ac6a4f6f7955c93639)

#### Facebook Login
**3. Log into Facebook on the displayed screen. (If you were already logged into Facebook beforehand, this screen will not appear.)**

![Facebook login screen](/liveagent/scripts/file.php?view=Y&file=acdbfa897b348a3b10e444373f7ae191)

#### Confirming Page Connection
**4. Close the English message screen and click "Connect" on the "Connect to Facebook Page" page.**

![Connection confirmation screen](/liveagent/scripts/file.php?view=Y&file=1299c73bba1487e13dec8ff76323da2d)

### Post-Integration Settings
#### Facebook Page Integration Complete
**5. The Facebook page has been integrated.**

![Integration complete screen](/liveagent/scripts/file.php?view=Y&file=13ea2004aad544fe3dcc0bd43290bf42)

#### Team Settings
**6. You can select the "Team" to assign tickets to by clicking "Edit" on the above screen.**

![Team settings screen](/liveagent/scripts/file.php?view=Y&file=83a1246ff062eacd3d7a3f547d5f1913)
