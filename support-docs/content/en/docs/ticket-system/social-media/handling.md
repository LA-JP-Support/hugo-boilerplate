## VIP Support Feature Overview
Yes, you can prioritize important customers with the VIP tag feature.

## How VIP Handling Works

### VIP Tag Features
- **VIP Tags**: Automatically or manually tag important customers
- **Priority Display**: Display inquiries from VIPs at the top
- **Dedicated Assignment**: Automatically assign to specific agents
- **SLA Priority**: Set shorter response deadlines for VIPs

### VIP Identification Methods
- Automatic determination based on follower count (when SNS integration is enabled)
- Determination based on past purchase amounts
- Manual VIP registration

## VIP Handling Use Case Scenarios

### Specific Use Cases
- Prioritize responses to influencer inquiries
- Premium support for high-value customers
- Rapid response to media personnel
