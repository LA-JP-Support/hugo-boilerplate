## Overview of SmartWeb Plans

### Plan Structure and Features

This section explains the SNS features (Facebook, Instagram, X, WhatsApp, Viber) available in each SmartWeb plan.

## Plan Details

### Basic Plan

- **SNS Features**: Not supported (can be added as optional feature)
- **Standard Features**: Ticket system, live chat, knowledge base, email integration

The Basic Plan is focused on essential customer support features and does not include SNS integration capabilities as standard. However, SNS features can be added for an additional fee to meet business needs. This plan is ideal for companies looking to build a support system centered around ticket management and live chat.

### Call Center Plan

- **SNS Features**: Not supported (can be added as optional feature)
- **Standard Features**: Phone & VoIP, IVR voice response, call recording, full basic features

The Call Center Plan is optimized for building support systems centered on phone support. While it comes standard with advanced calling features such as VoIP systems and IVR voice response, SNS integration is not included as standard. For companies that want to implement both phone support and SNS support, SNS features can be added as an optional feature.

### SNS/EC Enhanced Plan

- **SNS Features**: Full support (Facebook, Instagram, X, WhatsApp, Viber)
- **Standard Features**: Full basic features
- **Characteristics**: Plan specialized in SNS integration

The SNS/EC Enhanced Plan is designed for businesses that prioritize social media support. It provides complete integration with major SNS platforms including Facebook, Instagram, X (formerly Twitter), WhatsApp, and Viber as standard, allowing centralized management of inquiries from all SNS channels. This plan is ideal for companies aiming to merge digital marketing and customer support, or those using SNS as their primary communication channels.
Please note that LINE functionality within SNS features is available as an optional add-on.

### AI Chatbot Implementation Plan

- **SNS Features**: Not supported (can be added as optional feature)
- **Standard Features**: 24-hour AI support, full basic features

The AI Chatbot Plan focuses on AI technology that enables 24/7 automated response capabilities. This advanced plan optimizes human resources while enabling constant support, but does not include SNS integration features as standard. For companies that want AI to handle inquiries from SNS as well, SNS features can be added as an optional feature.
