## Supported Messaging Platforms

### WhatsApp
WhatsApp is a messaging app with a large global user base, and SmartWeb provides standard support for it. You can maximize WhatsApp's functionality for customer support and communication management. It enables real-time interactions between businesses and customers, achieving efficient dialogue.

### LINE
LINE is a particularly popular messaging platform in Japan. Currently, LINE support is available as a separate paid option, so please consult with our sales representative for specific implementation details. As it's an important communication channel for many Japanese companies, we offer customizable options.
