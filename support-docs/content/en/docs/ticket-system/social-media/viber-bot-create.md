## About Viber Bot Creation Status

### Latest Changes

Due to Viber's commercial model change on May 5, 2024, Viber bots can no longer be created by users and must be created by Viber partners or directly by Viber. This change has imposed significant restrictions on bot development for individuals and small businesses.

### Detailed Information

The restriction on Viber bot creation stems from the platform's goals to strengthen security and improve quality control. The main impacts include:

- Only approved partners can develop bots
- Improved bot quality and reliability
- Reduction of spam and malicious bots

### Current Options for Viber Bot Creation

Currently, there are two main ways to create Viber bots:

1. Participation in the Viber Partner Program
   - Bot development possible through official partnership
   - Requires a strict review process
   - Option for enterprises and large-scale organizations

2. Direct collaboration with Viber
   - Bot development with direct approval from Viber
   - Advanced features and integrations available
   - For large-scale business solutions

### Alternative Approaches

For individuals and small businesses, the following alternative strategies can be considered:

- Using existing approved bot templates
- Utilizing third-party bot development platforms
- Direct consultation with Viber's support team

For more details, please visit:
