## Keyboard Shortcuts Overview

LiveAgent offers various keyboard shortcuts to improve productivity. Below are the main shortcuts organized by function.

### Navigation Shortcuts

- **CTRL + H** - Navigate to the home tab
- **CTRL + ↑** - Move to the next open ticket tab
- **CTRL + ↓** - Move to the previous open ticket tab

### Ticket Management Shortcuts

- **CTRL + D** - Open "Tickets to Resolve" (if there are tickets to resolve)
- **CTRL + E** - Resolve ticket
- **CTRL + Q** - Close ticket window
- **CTRL + Left Click** - Open ticket in background (only from ticket list view)

### Message Operation Shortcuts

- **CTRL + I** - Display "Email Templates" list (when creating new ticket or replying to ticket)
- **CTRL + K** - Display "Frequently Used Phrases" list (when replying to ticket or handling chat)
- **CTRL + SPACE** - Display "Frequently Used Phrases" list (same as CTRL + K)
- **CTRL + S** - Send message

### Chat Shortcuts

- **CTRL + M** - Respond to chat
- **CTRL + SHIFT + M** - Respond to chat in background
