## Agent Rating Overview

A system that allows customers to evaluate agent responses. After support completion, customers can provide satisfaction feedback, and this data is used to visualize each agent's performance.

### Purpose and Significance of Rating

For example, when customers evaluate "response speed" and "courtesy" in customer surveys, improvement points and strengths become clear, which is useful for education and incentive design. Rankings also contribute to improving the motivation of the entire team.

### Key Features

**Points:**

- A system that allows customers to directly evaluate agents
- Visualize performance with quantitative data
- Clarify improvement points and utilize for education and training
- Ranking display that leads to improved motivation
- Available for Customer Satisfaction (CSAT) analysis
