## How to Connect LiveAgent with Gmail

### Gmail Connection in Cloud Version LiveAgent

#### Connection Steps
Cloud version LiveAgent accounts connect to Gmail accounts using Google's OAuth authentication. By authorizing this connection in a popup window, your Gmail mailbox will be connected with no further configuration required.

1. Log into LiveAgent and open the email account settings screen.
2. Select "Settings" > "Email" > "Mail accounts"
3. Click "Add mail account" and select the Gmail connector
4. Authorize your Gmail account in the popup window
5. Specify the mailbox "Team" and save

### Gmail Connection in Software Version LiveAgent

#### Basic Connection Steps
LiveAgent (software version) connects via IMAP/POP3 and SMTP servers using your Gmail username and password.

1. Log into LiveAgent and open the email account settings screen
2. Click "Add mail account" and select the Gmail connector
3. Enter your Gmail email address and password
4. Click the connect button

### Gmail Connection Requirements and Settings

#### Enabling IMAP Connection
1. Log into Gmail in your browser
2. Open "Settings" > "Forwarding and POP/IMAP" section
3. Enable IMAP access

#### Adjusting Security Settings
- Check Google account security notifications
- Allow access to your Google account
- If necessary, configure access for less secure apps

### Handling Password Changes

#### Reconfiguring Connection
When you change your Gmail account password:
1. Open "Settings" > "Email" > "Mail accounts"
2. Edit the relevant mail account
3. Update the connection with new authentication credentials

### Troubleshooting
If connection issues persist, please contact support.
