## Overview

There are differences in behavior between "Spam" in the ticket screen and the rule action "Mark as Spam".

## "Spam" in the Ticket Screen

### Ticket Processing
The ticket is treated as spam and excluded from the ticket list.

### Forum Post Deletion
For tickets created from forum posts, the posts are also deleted from the forum.

## Rule Action "Mark as Spam"

### Exclusion from Ticket List
Can be configured to exclude from the ticket list as spam.

### Forum Post Processing
Forum posts are not automatically deleted.

### Deletion Method
Deleted by completely deleting the ticket.
