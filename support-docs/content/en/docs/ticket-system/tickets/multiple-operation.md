## Agent Addition Overview

In the trial version, you can add agents (login IDs) from within the agent screen after logging in.

## Agent Addition Procedure

### Accessing the Agent Creation Screen

- From the left menu, click "Settings" > "Agents" > "Create Agent".

### Entering Agent Information

- Enter the agent's name and email address, and select their role and gender (adding a photo is optional).

### Completing Agent Creation

- Click "Create" and a notification email will be sent to the entered address. Once the recipient sets a password using the received email, the additional setup is complete.
