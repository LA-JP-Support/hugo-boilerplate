## File Attachment Overview

LiveAgent allows you to easily send and receive attachments in multiple formats. In business communication, the attachment feature plays an important role and enables smooth information sharing.

![](/liveagent/scripts/file.php?view=Y&file=58f42a50b31a9460a3cbe8058806d380)

## How to Attach Files

### Attachment Procedure
When sending emails from LiveAgent, click the "clip" icon at the bottom of the send form to attach files. You can easily attach files following these steps:

- Click the icon
- Select a file from your local device
- Confirm the file to be attached

### Saving Attached Files
Attached files are automatically saved within the related ticket, regardless of whether they are sent or received. This provides the following benefits:

- Easy history management
- Simple access to past files
- Easy communication tracking

## File Attachment Limitations

### File Size
The maximum file capacity that can be attached is 128 MB per send/receive operation. It supports large file sharing and is compatible with the following file types:

- Image files (JPG, PNG, GIF)
- Documents (PDF, DOCX, XLSX)
- Compressed files (ZIP, RAR)

This file attachment feature enables LiveAgent users to achieve efficient and flexible communication. It makes it possible to quickly share important information and materials.
