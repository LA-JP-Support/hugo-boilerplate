## Overview of Attachment Feature

The attachment feature is a convenient function that allows you to attach files to tickets. You can easily share images, audio, documents, and other files to supplement customer interactions.

### Benefits of Attachments

For example, when a customer reports a product issue and attaches a screenshot, agents can accurately understand the situation. This also expands the scope of responses, such as sending manuals as supplementary information for answers.

### File Format Control

You can specify the file formats that visitors can upload, such as "jpg, pdf, txt," controlling which types of files can be attached.

### Key Feature Points

- Ability to attach various files including images and documents
- Easy for customers to provide additional information
- Can also be used by agents to provide materials
- Reduces time to problem resolution
- Improves accuracy of interactions
