## Supported Devices and OS

LiveAgent supports Windows and Mac desktop/laptop computers as well as smartphones and tablets such as iPhone/iPad/Android, provided there is an internet connection.

### Desktop Environment

#### Windows
- Microsoft Windows 10 or later

#### Mac
- Apple Mac OS X

### Mobile Environment

#### Smartphones & Tablets

- Apple iOS
- Google Android

## Feature Limitations During Use

### Mobile Browser Limitations

- Chat reception is not available when using browsers on smartphones and tablet PCs. If you wish to use chat functionality, please use the dedicated apps mentioned below.

- In mobile view, only ticket viewing and processing are available. To use other features, you need to switch to desktop view.

- Phone call functionality is not available.

## Mobile Apps

LiveAgent provides free dedicated apps for iOS and Android. The app versions enable chat reception and response capabilities.

### App Downloads

- [iOS](https://itunes.apple.com/jp/app/live-agent/id661142245?mt=8)
- [Android](https://play.google.com/store/apps/details?id=com.qualityunit.liveagent)

### Main App Features

- Ticket viewing
- Ticket resolution, transfer, and reply
- Chat
- Ticket selection function (team)
- Viewing tags, notes, and teams
