## How to restore tickets marked as spam

Tickets marked as spam can be manually restored to the ticket list from the agent screen.

【Related items】　About automatic spam detection

### Ticket restoration procedure

1. Search in the agent screen

- From the left menu of the agent screen, select "Search" > "Tickets", then click "Custom search conditions"

![](/liveagent/scripts/file.php?view=Y&file=f24f4301a19ae08970cdefc258248520)

### Filtering spam tickets

- In the search conditions "Status", check only "Spam", then click "Apply"

### Restoring tickets

- Tickets excluded as spam will be displayed in the list

- After selecting the ticket you want to restore, click "Not spam" to restore the ticket to the list screen
