## Rewards & Badges Overview

Rewards & Badges is a gamification feature that distributes specific badges based on agent actions and achievements. It adds "recognition" to daily work, achieving both fun and growth.

### Badge Features and Purpose

For example, badges specialized in actions such as "Fastest Response," "Longest Chat," and "Most Tagging" are awarded, enhancing playfulness and motivation with unique titles. Badges are updated daily, can be customized, and can be combined with awards and performance evaluations. The fun of collecting badges contributes to improved agent retention and performance.

### Key Features and Benefits

**Key Points:**

- Automatic badge assignment based on actions such as chat/response count/break time
- Balance of fun and recognition through unique titles
- Customization available by administrators
- Daily updates, optimal for real-time motivation
- Contributes to team activation and agent retention
