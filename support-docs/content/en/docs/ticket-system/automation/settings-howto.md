## Overview
This explains how to add BCC addresses to emails in LiveAgent.

### BCC Addition Limitations
While there is no direct option to add BCC addresses to emails sent by agents from LiveAgent to users, equivalent functionality can be configured through "Rules". Ticket IDs and email content can be referenced using variables.
