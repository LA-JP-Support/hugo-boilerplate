## Overview of Invitation Chat

Invitation chat is a feature that displays invitation banners on the screen to offer chat support to site visitors who meet certain conditions, such as staying on a specified page for a certain amount of time.

![](/liveagent/scripts/file.php?view=Y&file=49b50b8dbbdfa5750a85bb1d1045f683)

## Conditions for Hiding Invitation Banners

When visitors decline the offer using the following methods, the invitation banner will not be displayed again unless they open their browser again and visit the specified page.

### Methods of Declining

1. Clicking on messages like "Not needed now" within the invitation banner
2. Clicking the "Ã—" in the upper right corner of the invitation banner to close it
3. Closing the browser

### Conditions for Re-display

If visitors navigate to another page without performing the above actions, the invitation banner will be displayed when they revisit the specified page.
