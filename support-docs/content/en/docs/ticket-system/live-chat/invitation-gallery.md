## Overview of Chat Invitation Gallery

A collection of chat invitation templates that can be automatically displayed based on conditions. Natural engagement can be achieved through triggers such as dwell time, exit intent, and page views.

By adjusting the wording and appearance of templates and implementing guided interactions tailored to page objectives, you can strengthen support on high-abandonment screens and high-value product pages.

## Key Features

### Flexible Operational Functions
- Select purpose-specific templates for immediate deployment
- Set trigger conditions based on dwell time and user flow

### Optimization Functions
- Improve response rates through A/B testing of messaging
- Prevent cart abandonment and exits to improve CVR

### Multi-Purpose Applications
- Effective for both support and sales purposes
