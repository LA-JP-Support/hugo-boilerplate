## What is a Search Widget

A feature that allows you to embed a search bar on your website that displays real-time search results for knowledge bases, forums, suggestion boards, and more.

### Features of Real-time Search

Suggestions are displayed as users type, allowing smooth access to desired articles and FAQs. It's also possible to install multiple widgets with narrowed search scopes.

## Main Benefits

The intuitive search experience helps improve customer satisfaction and reduce the number of inquiries.

### Key Functions of Search Widgets

- Real-time search targeting articles, forums, suggestions, and more
- Provides embeddable search bars for web pages
- Improves accuracy and convenience of site search
- Multiple widget installation possible by category
- Smoothly supports customers in achieving their goals
