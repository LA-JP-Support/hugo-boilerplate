## How to Use Chat Support from Mobile Devices

### Conditions for Starting Chat
Customers using mobile devices such as smartphones and tablets can also request chat support if they have a browser with JavaScript execution enabled.

![](/liveagent/scripts/file.php?view=Y&file=2b67b0466765e073d25a242bfba87548)

### Chat Operations on Smartphones

When you tap the chat button in a smartphone browser, a chat window optimized for mobile device screen sizes will open. You can communicate in real-time with LiveAgent agents, similar to using a messaging app.

### How to End a Chat

To end a chat from the customer side, tap the menu in the upper right corner of the chat window and tap "End Chat."
