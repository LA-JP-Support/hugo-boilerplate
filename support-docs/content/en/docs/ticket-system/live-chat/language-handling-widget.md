## Overview of Language Adaptable Widgets

A feature that allows you to configure chat, contact, and feedback widgets for multilingual support. It optimizes appearance and behavior according to user languages and enhances customer support in multicultural environments.

## Key Features of Multilingual Support

### Language-Specific Widget Display
Widget display can be customized for each language, allowing Japanese widgets to be shown to Japanese users and English widgets to English users. Language-specific department assignment is also possible, improving response accuracy and user satisfaction. The convenience provided by multilingual support is a significant advantage for global companies and websites targeting multilingual customer bases.

## Main Functions and Benefits

### Detailed Multilingual Support Features
- Chat and input forms can be configured in multiple languages
- Automatic display switching based on user language
- Agent department assignment possible for each supported language
- Maintains UI consistency required for global support
- Contributes to improved customer experience and differentiation
