## What are "Frequently Used Phrases"

"Frequently Used Phrases" is a feature that allows you to call up pre-registered text using keywords. By registering phrases frequently used in email and chat responses as templates, you can eliminate the need to type the same phrases or sentences repeatedly, reducing response time, preventing typing errors, and improving operational efficiency.

## How to Use "Frequently Used Phrases"

### How to Call Up Frequently Used Phrases

- With the cursor positioned in the text input field of the agent's chat screen, press **"Ctrl key + Space key"** to call up the "Frequently Used Phrases" list.

　![](/liveagent/scripts/file.php?view=Y&file=3b5f13debf345ee09fb2ea04f9f872c8)

### Selecting and Searching Phrases

- When you select a message from the list, it will be reflected in the text input field of the agent screen.

- To search for "Frequently Used Phrases," display the "Frequently Used Phrases" list and enter a search term of 2 or more characters in the text input field. LiveAgent will suggest appropriate phrases.

　　![](/liveagent/scripts/file.php?view=Y&file=bded6e2e3d55fa0a5a326b6ca978d403)

## How to Set Up "Frequently Used Phrases"

### Setup Procedure

- From the agent screen left menu, select Settings > Automation > Frequently Used Phrases

- Click "Create Message" and register additional phrases on the following screen

### Message Sharing Settings

- For messages, specify one of "Personal use only," "Team use," or "Global use"
