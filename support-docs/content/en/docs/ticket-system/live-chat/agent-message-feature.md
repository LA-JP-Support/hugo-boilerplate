## Overview of Message Feature

When no agents are available to handle live chat, such as outside business hours or when agents are away, customers can leave messages through the chat button.

## Message Feature Settings

### Enabling the Message Feature

For the setup procedure of the message feature that displays the chat button when offline, please refer to "How to Set Up Chat Buttons".

### Disabling the Message Feature

To disable the message feature when agents are unavailable, go to the agent screen left menu and select Settings > Chat > Chat Button > Edit > Offline Button, then uncheck "Use offline button". The chat button will be hidden when offline.
