## What is Invitation Chat

Invitation chat is a feature that displays a screen on the web to encourage chat conversations with site visitors who meet certain conditions, such as staying on a web page for a certain amount of time. The invitation screen can be customized using templates and original images.

*The chat feature is available in two plans: [「Professional」and「Advanced」](http://liveagent.jp/plan.html).

## How to Set Up Invitation Chat

### Selecting Chat Invitation Button Template

From the agent screen left menu, select "Settings" > "Chat" > "Invitation" and click "Create".

Choose the button format you want to use from the sample screen. Click "+ Select Button" or "Select".

Buttons can be selected in different types depending on their display position. If you want to use original images, select "Custom" for the button style.

### Customizing Chat Invitation Screen

This is the screen displayed when inviting site visitors to chat. Specify images, messages, colors, etc.

### Customizing Chat Window

This is the screen where site visitors click on the invitation screen and actually engage in chat conversations. Specify title, welcome message, screen design, display position, colors, etc.

### Setting Invitation Chat Display Conditions

First, specify "Page stay time" between 5 seconds to 30 minutes.

Next, set the following conditions by combining them with and/or:
- URL
- Referrer URL
- Whether the visitor is new or returning

### Invitation Chat General Settings

Select the invitation screen name, assigned team to receive chats, and language. By turning on the "Active" checkbox, the invitation chat feature will operate (unchecking it will stop the feature).

*Once invitation chat is implemented, statistics (views, accepts, click-through rate) will be displayed on this page.

### Website Integration

When "Integration" is displayed, code for integrating with your website will be shown.

Please paste the displayed code directly into your website pages. Place the code before `</body>` on the pages you want to use.

- Buttons and pages cannot be displayed in local files, so please make sure to upload to a web server.
