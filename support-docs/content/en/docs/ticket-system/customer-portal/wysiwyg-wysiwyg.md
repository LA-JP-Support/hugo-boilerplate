## WYSIWYG Editor Overview

A "what you see is what you get" editor for knowledge bases that makes article editing simple for everyone. Create high-quality articles efficiently without specialized knowledge.

### Main Features

For knowledge base article creation, the following features are provided:

- Intuitive formatting of images and text
- Flexible use of HTML/CSS
- Visual article editing
- Real-time preview

## Benefits

### Intuitive Editing Experience

For example, anyone can easily create new FAQ articles with images or visually arrange links and headings. Since you can edit while viewing the preview, high-quality articles can be published quickly.

### Efficient Information Sharing

Lowers the barrier to document creation, achieving both speed and quality in information sharing simultaneously.

## Key Features

### Ease of Use

- What you see is what you get editing that's manageable in daily work
- Intuitive image, link, and style adjustments

### Advanced Editing Functions

- Supports HTML and CSS code editing
- Effective quality control through preview-while-editing

### Enhanced Team Productivity

- Non-technical members can also participate in article creation
