## Search Widget Overview

This section explains how to create a widget (search box) for searching article content in your customer portal. This widget allows customers to quickly find the answers they need through keyword searches instead of submitting inquiry messages.

## Search Widget Creation Steps

### Step 1: Create New Search Widget
Click the "Create" button in "Customer Portal" > "Search Widget".

![](/liveagent/scripts/file.php?view=Y&file=465025f1fe85260c9913e422e09ac5c3)

### Step 2: Basic Configuration
Configure the following items in the "Overview" section:
- Select knowledge base category
- Specify language
- Enter widget name
- Optional settings
  - "Open articles in new window"
  - "Active" checkbox

![](/liveagent/scripts/file.php?view=Y&file=e1cf2cdc148346cf0e5056a5a470d93b)

### Step 3: Design Customization
You can customize the following items in the "Design" section:
- Title
- Search box hint text
- Color settings
- Custom CSS

![](/liveagent/scripts/file.php?view=Y&file=cbddf8013f3205e9f852dec0ab4ee143)

### Step 4: Web Page Integration
Insert the provided code just before the </body> tag on your web page.
