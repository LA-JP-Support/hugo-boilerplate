## LiveAgent's Multi-Language Support

LiveAgent supports over 30 languages including Japanese, English, and Chinese. Language display for each function needs to be configured separately for each feature.

### Default Language Settings

Set the default language for knowledge base skins (menu display, etc.) and other functions.

【Related Items】Regional and Language Settings

### Language Customization for Each Function

#### Live Chat

You can individually configure the language of the chat button and system messages within the chat window. For text input fields, you can enter and display any custom text strings.

【Related Items】How to Configure Chat Button Settings

#### Contact Forms

You can individually configure the language of system messages within forms. For text input fields, you can enter and display any custom text strings.

【Related Items】Contact Form Overview and Usage Instructions

#### Customer Portal

The language of side menus and system messages follows the "default language settings" configuration.

##### Knowledge Base

To manage knowledge base article content by language, we recommend setting up categories for each language and creating article content under those categories.

For search widgets, you can configure the system display language.

【Related Items】Building a Customer Portal

### Agent Interface Language Settings

The agent interface language can be freely selected from the language selection dropdown menu on the login screen, regardless of the "default language" settings.
