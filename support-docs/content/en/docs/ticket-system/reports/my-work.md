## Work Report Overview

### Report Display Feature
Logged-in agents can view their own customer service reports anytime through graphs and tables.

### Report Customization
In addition to time periods, you can add channels, teams, and tags to search conditions to create detailed reports.

## Report Analysis Features

### Performance Visualization
This report allows you to compare the number of cases handled per hour and customer evaluations at a glance.

### Detailed Work Analysis
You can also check detailed work timesheets, making it ideal for analyzing the performance of your daily work.
