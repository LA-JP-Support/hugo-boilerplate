## Performance Report Overview

Performance Report is a feature that comprehensively evaluates system-wide and agent activities. You can measure response speed, resolution rate, processing volume, and more.

## Main Functions and Purpose

### Performance Metrics Measurement

By quantifying achievements in daily operations, you can discover improvement opportunities and success stories. For example, you can reference teams with particularly excellent response speeds to help improve other teams.

### Data-Driven Decision Making

Data-driven decisions can elevate overall performance across the board.

## Key Points

### Measurable Metrics
- Measure response speed, resolution rate, and processing volume

### Business Improvement Perspective
- Identify improvement opportunities and success stories
- Share best practices through team comparisons
- Support business improvement with quantitative evaluation
- Elevate overall performance
