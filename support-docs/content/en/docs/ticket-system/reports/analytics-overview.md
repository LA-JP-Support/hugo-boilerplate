## Overview of Analytics Overview

Analytics Overview is a dashboard feature that provides an at-a-glance view of overall system performance. It aggregates key metrics such as ticket count, response time, and SLA achievement rate, making it useful for everyone from executives to frontline leaders.

### Main Features

For example, administrators can check the dashboard to understand which time periods have concentrated inquiries and optimally allocate personnel. It's also possible to discover business processes that need improvement by analyzing long-term trends.

### Data Display and Decision Making

Data is displayed visually, making even complex numbers intuitively understandable and supporting rapid decision-making.
