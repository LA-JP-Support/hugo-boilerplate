## Introduction
**While it is technically possible to use LiveAgent and FlowHunt directly without utilizing SmartWeb's construction services**, there are several important points we would like you to understand.

## Direct Use of LiveAgent

### Service Overview
LiveAgent is a global helpdesk software provided by Quality Unit and is available for use worldwide.

### Direct Application
- You can apply directly through the official website (LiveAgent.com)
- A 30-day free trial is available
- Basic support will be provided in English

### Using Through Interwork
- **〈Important〉** By applying through this URL ([https://www.liveagent.com/#interworkcorp](https://www.liveagent.com/#interworkcorp)), we can provide **free Japanese support**
- Setup assistance and training during implementation are offered as separate paid services

## Direct Use of FlowHunt

### Service Overview
FlowHunt is also a Quality Unit product that can be used directly, but:

- **All support will be provided in English**
- Building AI chatbots requires a certain level of technical knowledge

## About Interwork
Interwork Corporation is the general agent for Quality Unit products in Japan.

## Challenges with Self-Implementation

### Technical Challenges
- Integration setup between LiveAgent and FlowHunt
- Building knowledge bases and preparing learning data
- Integrating chat widgets into websites

### Operational Challenges
- FlowHunt offers English support only
- Quick response during troubleshooting
- Continuous tuning and maintenance
