## Integration with LiveAgent Ticket System

AI Answer Composer is **fully integrated with LiveAgent's ticket management system**. Support agents can seamlessly utilize AI functionality within their ticket handling workflow.

## How the Integration Works

### Automatic Integration from Incoming Messages

1. Inquiries arrive via email or contact forms
2. LiveAgent automatically creates tickets
3. AI analyzes inquiry content and generates response drafts
4. Response drafts appear as notes in the ticket screen

### Operations within Ticket Screen

Support agents can perform the following operations within their regular ticket handling workflow:

- Review response drafts
- Edit as needed
- Enhance with AI Answer Improver
- Send replies

**All operations are completed without leaving the ticket management screen**.

## Benefits

### Efficient Workflow

Since AI functionality is naturally integrated into existing ticket handling workflows, support agents don't need to learn new tool operations.

### Centralized History Management

All interactions are recorded within tickets, making it easy to manage response histories.

### Enhanced Team Collaboration

Seamlessly integrates with existing LiveAgent features such as ticket assignment, escalation, and comment addition.

## Available Features

### AI Answer Composer

Automatically generates response drafts from conversation history and knowledge base, displaying them in the ticket screen.

### AI Answer Improver

Allows enhancement of generated response drafts within the ticket reply editor.

### Automation Rules

Automatically switches the knowledge that AI references based on incoming email recipients or content.

## Setup

To use AI Answer Composer, the following configuration is required:

1. LiveAgent account setup
2. FlowHunt integration configuration
3. Knowledge base preparation

For detailed setup instructions, please contact our support team.
