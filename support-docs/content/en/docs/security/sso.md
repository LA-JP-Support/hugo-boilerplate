## What is Single Sign-On (SSO)

### Basic Concept of SSO
An authentication feature that leverages authentication providers such as Google, Microsoft, Facebook, and Okta to enable access to multiple systems including LiveAgent with a single login. Achieves a high-level balance between security and convenience.

## Benefits of SSO

### Simplified Login Process
Implementing SSO eliminates the hassle of remembering and managing multiple passwords, streamlining the login process. Users can access multiple applications with a single secure authentication, which aligns with corporate security policies.

### Enhanced Security
Furthermore, when combined with two-factor authentication, even stronger security can be achieved.

## Key Features of SSO

### Supported Providers
- Microsoft
- Google
- Facebook
- Okta

### Main Benefits
- Multiple systems accessible with a single ID
- Balances login convenience and security
- Enhanced defense capabilities through integration with multi-factor authentication
- Contributes to reduced management costs and improved user experience
