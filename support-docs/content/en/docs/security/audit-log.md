## What is an Audit Log

A function that records all operations performed within the system. It allows you to check who performed which operations and when, helping with security management and problem resolution.

### Key Features of Audit Logs

For example, all critical actions such as ticket deletion and configuration changes are recorded, enabling identification of responsibility when troubles occur. It is also important for strengthening compliance and internal controls.

## Important Points of Audit Logs

### Recorded Information

- Automatically records all operation history
- Enables cause tracking when troubles occur
- Helps prevent unauthorized use and mistakes
- Essential function for compliance strengthening
- Makes it easier for administrators to understand situations
