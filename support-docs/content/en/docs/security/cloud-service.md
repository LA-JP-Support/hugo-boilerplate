## Service Delivery Model

SmartWeb is provided **as a cloud service (SaaS) only**.

| Delivery Model | Availability |
|----------|----------|
| **Cloud Edition (SaaS)** | ✓ Available |
| **On-Premise Edition** | ✗ Not Available |

## Benefits of Cloud Edition

### Get Started Immediately

| Benefit | Description |
|----------|------|
| **Instant Usage** | No server setup required, available immediately after registration |
| **Automatic Updates** | Always access the latest features |
| **Scalability** | Automatically scales based on usage |

### Reduced Operational Burden

| Benefit | Description |
|----------|------|
| **No Server Management** | No need for hardware or OS management |
| **No Maintenance Work** | Patching and backups are automatic |
| **High Availability** | Redundant configuration in data centers |

## Data Centers

Even with the cloud edition, you can choose where your data is stored.

| Region | Location |
|-----------|------|
| **Asia** | Singapore (recommended for Japanese customers) |
| **US** | United States |
| **EU** | Europe |
