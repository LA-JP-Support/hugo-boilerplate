## Automatic Updates

Since SmartWeb is a cloud service (SaaS), **updates are automatically implemented**.

| Item | Details |
|------|---------|
| **Version Management** | Not required (automatic) |
| **Security Patches** | Immediate application |
| **New Feature Addition** | Automatic reflection |
| **Downtime** | Minimal |

## Benefits of Automatic Updates

| Benefit | Description |
|---------|-------------|
| **Always Up-to-Date** | Access to the latest features and security |
| **Zero Operational Load** | No server management or maintenance required |
| **Enhanced Security** | Immediate response to vulnerabilities |
| **Improved Stability** | Automatic implementation of performance improvements |

## Checking Update Information

Update details can be checked at the following locations:

| Information Source | URL |
|-------------------|-----|
| **Official Changelog** | [changelog.liveagent.com](https://changelog.liveagent.com/) |

### Available Information

- Addition of new features
- Improvements to existing features
- Security enhancements
- Bug fixes

## Scheduled Maintenance

When scheduled maintenance is required, it will be announced in advance on the status page.

**Status Page**: [https://status.liveagent.com/](https://status.liveagent.com/)
