## Uptime Target

SmartWeb operates with a target of **99.9% or higher monthly uptime**.

| Item | Details |
|------|---------|
| **Target Uptime** | 99.9% or higher |
| **Measurement Period** | Monthly |
| **Performance** | Consistently achieving target |

## Mechanisms for Achieving High Uptime

| Measure | Description |
|---------|-------------|
| **Redundant Configuration** | Multiple server setup in data centers |
| **Automatic Failover** | Automatic switching during failures |
| **24-Hour Monitoring** | Continuous system status monitoring |
| **Regular Backups** | Safe data storage |

## Real-time Status Monitoring

System operational status can be checked in real-time.

**Status Page**: [https://status.liveagent.com/](https://status.liveagent.com/)

### Available Information

- Current system operational status
- Status of each service component
- Past incident history and response details
- Scheduled maintenance plans
