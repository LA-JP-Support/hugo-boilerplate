## What is an Agent

Support staff who use LiveAgent are called "agents." If two or more people will be using the system, please add multiple agents.

### Agent Permission Levels

You can assign the following permissions to added agents:

- **General Agent**: Handle tickets (role of responding to customers)

- **Administrator**: In addition to agent functions, can access system settings and view reports

- **Owner**: In addition to administrator functions, can change service plans and access billing information

## How to Create an Agent

### Steps to Add an Agent

From the left menu of the agent screen, select Settings > Agents > Create Agent. Enter the name and email address, then click the "Create" button. Login information will be sent to the registered email address.
