## Email Notifications Overview

The email notification feature is a system that informs agents of new tickets and updates in real-time.

### Main Features

A feature that sends email notifications for new tickets and updates. Agents can stay constantly informed of the latest status and won't miss important cases.

### Customizable Notification Settings

Notification content can be customized, and settings can be configured to receive only necessary information. This prevents information overload while enabling efficient response handling.

## Key Features

### Automatic Notification System
- Automatic notifications for new tickets and updates
- Quick response without missing important cases

### Flexible Settings
- Customizable notification content
- Supports agent work efficiency
- Prevents information overload and acquires only necessary information
