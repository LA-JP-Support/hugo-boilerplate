## Notification Sound Settings Overview

A feature that allows you to customize notification sounds played within LiveAgent. Audio alerts for new tickets and updates help prevent missed responses.

You can also disable notification sounds or set custom sounds, allowing flexible adjustment to match your work environment and personal preferences.

## Key Features of Notification Sound Settings

### Real-time Notifications
- Instantly recognize new tickets and updates through audio alerts

### Customization
- Customizable sound types
- Can be disabled for quiet work environments

### Business Efficiency
- Support for preventing missed responses
- Adaptable to agent work styles
