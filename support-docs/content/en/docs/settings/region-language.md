## Region and Language Settings

### Regional Settings Overview

For all languages, appropriate date and time formats for each language are defined in the system. To change the date and time format, select "Use custom date, time, and number formats" to configure.

### How to Customize Date and Time Display

- Select "Settings" > "System" > "Language" from the left menu of the agent screen

- Turn on the radio button for "Use custom date, time, and number formats" in "Regional Settings" to display the configuration options

- Configure the formats for "Date", "Time", and "Numbers (symbols used for thousands separator and decimal point)", then click "Save".

## System Language Management

### Default Language Settings

You need to select one language from the languages registered in LiveAgent as the system's default language.

#### Steps to Change Default Language

- To change the default language, select "Settings" > "System" > "Language" from the left menu of the agent screen

- From the displayed list of languages, click "Set as Default" next to the language you want to set as default

### Adding and Managing Languages

#### Importing Language Files

The language that agents use on LiveAgent is determined by the language selected on the agent login screen.
