## About Periodic Report Generation

### Report Creation Availability

Yes, by using SmartWeb's report feature, you can automatically generate daily, weekly, and monthly reports and have them delivered via email. This feature supports efficient information management and decision-making for your business.

### Report Creation Details

#### Report Frequency
You can flexibly choose the report generation frequency:
- Daily reports: Understand detailed daily business conditions
- Weekly reports: Check weekly progress and trends
- Monthly reports: Utilize for long-term strategic planning

For example, customer support teams can use daily reports to analyze customer service quality, while marketing teams can use monthly reports to analyze advertising effectiveness.

#### Delivery Method
Reports are automatically delivered to specified email addresses, eliminating the need for manual work.
- Multiple recipients can be configured
- Sent in PDF or tabular data format
- Easy schedule configuration

#### Usage Requirements
To use the report feature, you need to upgrade to a compatible plan.
- Available on Business plan and above
- Provided as a standard feature with no additional cost
- Highly scalable customization options

For details about each plan, please check SmartWeb's official website.
