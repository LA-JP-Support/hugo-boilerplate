## How to Update the Knowledge Base

### Easy Article Management with WYSIWYG Editor

You can easily add and edit articles directly from your web browser. The WYSIWYG editor allows for intuitive operation similar to Microsoft Word, with no HTML knowledge required.

Specifically, the following operations are possible:
- Text input and formatting
- Inserting images and videos
- Adding links
- Creating tables and bullet lists

### Advanced Operational Features

Category creation, display order, and access restrictions for internal-only knowledge can all be managed with just one click.

Key advanced features include:
- Category management: Group related articles together
- Access permission settings: Publish only to specific users or departments
- Article reordering: Adjust display order based on importance or update date
- Version control: Retain and restore previous article versions

These features enable knowledge sharing within organizations and efficient information management. With intuitive operations, anyone can easily update and manage the knowledge base.
