## System Requirements (End User)

The following environments are supported for browsing web pages that incorporate LiveAgent functionality. Please check the recommended environments to use the latest communication features safely and smoothly.

### PC Environment

#### Supported OS
- Windows: Windows 10 or later (latest version recommended)
- Mac OS: Latest macOS
- Linux: Major distributions such as Ubuntu, Fedora

#### Supported Browsers
- Edge: Latest version recommended
- Chrome: Latest version
- Safari: Latest version
- Firefox: Latest version

Since LiveAgent utilizes the latest web technologies, we recommend using the latest browsers at all times. Older browsers may not properly support advanced features such as chat functionality and screen sharing.

**Note:** Internet Explorer support has ended. Chat widgets and other features will not function properly.

### Mobile Environment (Smartphone/Tablet)

#### Supported OS
- iOS: iOS 13 or later
- Android: Android 9.0 or later

#### Supported Browsers
- Safari: iOS default browser
- Chrome: Android default browser
- Firefox: Latest mobile version

In mobile environments, we provide an optimized interface with responsive design that adapts to screen size and communication conditions. Smartphones and tablets deliver the same comfortable operability as PCs.

For network connectivity, we recommend a stable internet connection (Wi-Fi or 4G/5G). Slow or unstable connections may affect the quality of chat functionality and screen sharing.
