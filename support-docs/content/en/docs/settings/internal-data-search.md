## Overview

Search items viewable from the LiveAgent agent interface. If you click the "Search" button without specifying search conditions, all data will be displayed as search results.

![](/liveagent/scripts/file.php?view=Y&file=2865d5b6d27a7113ebed0c83579fe425)

## Search Target Details

### Ticket Search

Full search of tickets stored in the system is possible. From the agent interface left menu, display "Search" > "Tickets" and search using "Keywords" and "Custom Search Conditions" (ticket status, source, tags, etc.).

### Contact Search

Full search of "Contacts" (customer information) stored in the system is possible. From the agent interface left menu, display "Search" > "Contacts" and search using "Keywords".

### Screen Search

A sitemap-like function that searches all LiveAgent system screens. From the agent interface left menu, display "Search" > "Screens" and search using "Keywords".

### Knowledge Base Search

Full search of the "Knowledge Base" built on LiveAgent is possible. From the agent interface left menu, display "Search" > "Knowledge Base" and search using "Keywords".

### Event Log Search

Search of system event logs is possible. From the agent interface left menu, display "Settings" > "Tools" > "Event Log" and search using "Keywords" and "Custom Search Conditions" (ticket status, source, tags, etc.). Event log clearing is also possible.

### Outbox Search

Search of emails sent from LiveAgent is possible. From the agent interface left menu, display "Settings" > "Tools" > "Outbox" and search using "Keywords" and "Custom Search Conditions" (ticket status, source, tags, etc.). Deletion and resending of sent emails is also possible.
