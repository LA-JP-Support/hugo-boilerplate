## What are Canned Messages / Macros

A feature that allows you to template frequently used responses and insert them instantly. It maintains consistency in responses and improves response speed.

For example, for common questions like "how to reset password," you can prepare canned messages to respond within seconds. It's also possible to execute multiple operations in bulk as macros.

## Key Features

### Benefits of Templating

- Template frequently used responses
- Insert responses with one click
- Improve response consistency and speed

### Feature Extensibility

- Automate multiple operations with macros
- Reduce agent workload
