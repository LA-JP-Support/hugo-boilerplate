## Multi-Brand Support Overview
Yes, you can manage support operations for multiple brands with a single account.

## Multi-Brand Management Features

### Department Management
- **Department Function**: Independent inquiry management for each brand

### Customization Features
- **Customer Portal**: Set different designs and URLs for each brand
- **Email Templates**: Use different signatures and logos for each brand

## Use Case Scenarios

### Usage Examples by Business Type
- Operate multiple e-commerce sites and handle each as a separate brand
- Use a shared system between parent and subsidiary companies
- Separate service windows for B2B and B2C products

## Reporting

### Analytics Features
- Brand-specific inquiry analysis
- Cross-brand reporting is also possible

## Usage Conditions
Available at no additional cost.
