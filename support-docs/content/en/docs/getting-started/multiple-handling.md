## Support for Multiple Branches and Stores

Yes, we support usage across multiple branches and stores.

### Department Management

You can create "departments" for each store or branch and automatically route inquiries accordingly.

### Usage Scenarios

#### Flexible Inquiry Management
- Independent inquiry management for Tokyo headquarters, Osaka branch, and Fukuoka branch
- Separate chat buttons for Store A and Store B
- Department-specific business hours and holiday settings

### Report Analysis

#### Detailed Performance Visualization
- Number of inquiries by branch/store
- Response time comparison analysis
- Company-wide reports also available

### Implementation Cost

Setting up multiple departments is available at no additional charge.
