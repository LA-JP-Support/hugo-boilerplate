## Mobile App Overview

With the LiveAgent mobile app, you can **handle tickets and respond to chats on the go**.

| Supported OS | Device | Download |
|--------------|--------|----------|
| **iOS** | iPhone / iPad | App Store |
| **Android** | Smartphone / Tablet | Google Play |

## Main Features

### Ticket Management

| Feature | Description |
|---------|-------------|
| **Ticket Handling** | View, reply to, and resolve new tickets |
| **Ticket Transfer** | Transfer to other agents or departments |
| **Filters** | Organize tickets with preset filters |
| **Tags & Notes** | View tags and agent notes |

### Real-time Notifications

| Feature | Description |
|---------|-------------|
| **Push Notifications** | Instantly receive alerts for new inquiries |
| **Chat Notifications** | Receive customer messages in real-time |
| **Status Updates** | Get notified of ticket status changes |

### Chat Support

| Feature | Description |
|---------|-------------|
| **Live Chat** | Real-time chat with customers |
| **Chat Responses** | Reply instantly even when away from office |
| **Chat History** | Review past conversations |

## Mobile App Benefits

| Benefit | Description |
|---------|-------------|
| **Location Independent** | Provide customer support outside the office |
| **Instant Response** | Start responding quickly with push notifications |
| **Business Continuity** | Maintain support quality while traveling or away |
| **Prevent Opportunity Loss** | Avoid customer churn due to delayed responses |

## Usage Scenarios

- **Emergency Response While Out** - Respond immediately to urgent inquiries
- **Remote Work** - Support operations from home or cafes
- **During Sales Visits** - Make effective use of travel time
- **At Event Venues** - Provide on-site customer support

## Download Instructions

### iOS (iPhone / iPad)

Search for "LiveAgent" on the App Store and download

### Android

Search for "LiveAgent" on Google Play and download

1. Download and install the app
2. Log in with your LiveAgent account
3. Allow push notifications
4. Start using
