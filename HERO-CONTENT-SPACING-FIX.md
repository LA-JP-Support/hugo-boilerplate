# ヒーローセクションと本文の間隔調整

## 🎯 修正内容

赤枠部分（ヒーローセクションと本文の間）の余白が大きすぎたため、以下を調整しました。

---

## 📊 変更点

### 1. ヘッダー下部のボーダーを削除

**Before:**
```html
<header class="py-12 sm:py-16 border-b border-gray-200">
```

**After:**
```html
<header class="py-12 sm:py-16">
```

**理由：**
- ヘッダーとコンテンツの分離が強すぎた
- 区切り線を内部に移動

---

### 2. 区切り線を薄く調整

**Before:**
```html
<div class="my-8 border-t border-gray-200"></div>
```

**After:**
```html
<div class="mt-6 mb-4 border-t border-gray-100"></div>
```

**変更内容：**
- 上余白：`my-8` (2rem) → `mt-6` (1.5rem)
- 下余白：なし → `mb-4` (1rem)
- ボーダー色：`border-gray-200` → `border-gray-100`（より薄く）

---

### 3. 本文の上部余白を削減

**Before:**
```html
<div class="prose ... py-12 sm:py-16">
```

**After:**
```html
<div class="prose ... pt-6 pb-12 sm:pt-8 sm:pb-16">
```

**変更内容：**
- 上余白：`py-12` (3rem) → `pt-6` (1.5rem)
- 下余白：変更なし（`pb-12`を維持）
- デスクトップ：`sm:py-16` → `sm:pt-8 sm:pb-16`

---

## 💡 デザイン原則

この調整は以下の原則に基づいています：

1. **視覚的階層**
   - 重要な要素（ヘッダー）の後は適度な余白
   - 過度な余白は不要

2. **読みやすさ**
   - 余白が多すぎると情報が分散
   - 適度な余白で情報をグルーピング

3. **美的バランス**
   - 余白の比率：1.5 : 1 : 1.5
   - 調和のとれたレイアウト
