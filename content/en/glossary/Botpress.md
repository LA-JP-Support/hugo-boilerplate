## What is Botpress?

**Botpress is a comprehensive platform for building AI-powered chatbots and conversational agents. It features visual flow design, advanced [NLP](NLP.md), and direct integration with [LLMs](LLM.md) like GPT-4 and Claude. For developers, it offers API and SDK customization options; for non-technical users, it's completely no-code.**

> **In a nutshell:** The "AI chatbot workshop" where you can assemble cutting-edge AI technology visually.

**Key points:**

- **What it does:** LLM-powered AI chatbot building platform
- **Why it matters:** Implements advanced AI features with minimal development complexity
- **Who uses it:** Large enterprise customer service teams and startups

## Key features

**Visual flow editor**
Drag-and-drop conversation logic design with intuitive handling of complex conditional branches.

**[LLM](LLM.md) integration**
Built-in support for GPT-4, Claude, Gemini, and other cutting-edge models, with custom API key support.

**[RAG](RAG.md) (Retrieval-Augmented Generation)**
Automatically searches the knowledge base for relevant information, preventing [hallucinations](Hallucination.md) (false information generation).

**Omnichannel deployment**
Deploy simultaneously to Web, WhatsApp, Slack, Telegram, and 10+ other channels.

**Enterprise security**
SOC II and GDPR compliant, with self-hosting options available.

## How it works

Using Botpress starts with designing conversation flows. Assemble visual editor blocks to create paths that respond to user intent. Then configure LLM settings to enable natural conversation. Connect a [knowledge base](Knowledge-Base.md) to automatically generate answers from documents.

After design, deploy to multiple channels at once. Post-deployment, use an [analytics dashboard](Analytics.md) to analyze conversation patterns and make continuous improvements.
