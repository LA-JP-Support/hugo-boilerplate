## What is Remote Support?

**Remote Support is a service where support personnel connect to a user's computer or device over the internet and provide troubleshooting and technical assistance from a remote location.** For example, when a computer slows down, if the user consents, a support representative can view the computer's screen and use the mouse and keyboard to solve the problem without requiring a physical visit.

> **In a nutshell:** A service where "your computer is broken, but you don't need to call someone—they can fix it from far away."

**Key points:**

- **What it does:** Connect to a user's device over the internet and resolve issues remotely
- **Why it matters:** Onsite support is time-consuming and costly; remote response is the key to efficiency
- **Who uses it:** IT companies, help desks, customer support teams, ISPs, SaaS companies

## Why It Matters

Onsite support takes time and costs money. Remote support can respond within minutes and achieves 70% cost reduction and 90% reduction in response time.

Especially since [COVID-19](COVID-19.md), remote work has increased, and demand for remote support has grown rapidly. Many companies now operate 24/7/365 remote support services.

## How It Works

The remote support process has three stages.

**Connection:** The user obtains an access code from the support website or the support team sends an invitation link. Secure [VPN](VPN.md) or [RDP](RDP.md) (Remote Desktop Protocol) connections are established using encryption.

**Diagnosis:** With the user's permission, the support representative reviews system logs, performance metrics, and software settings to identify the root cause. This progresses collaboratively with the user via [chat](Chat.md) or voice calls.

**Resolution:** The representative implements necessary fixes (software updates, configuration changes, malware removal, etc.) and verifies the solution with the user before ending the session.

## Real-World Use Cases

**Corporate IT Support**

When employees report password issues or software installation problems, the help desk responds immediately, resolving issues within minutes.

**Customer Support**

Software makers' support teams connect to user computers to provide complete troubleshooting and configuration assistance.

**Scheduled Maintenance**

Companies periodically connect to customer systems to implement security updates and performance adjustments.

## Benefits and Considerations

Remote support achieves 90% faster response times, 70% cost reduction, and 24/7 availability. Users also benefit from not having to wait for someone to visit them.

The primary concerns are [security](Security.md) and [privacy](Privacy.md). Since access is possible to a user's entire computer, choosing a trusted vendor is critical, with data encryption and "operation logs" being essential.

## Frequently Asked Questions

**Q: Could my data be stolen during remote support?**

A: Trusted services ([TeamViewer](TeamViewer.md), [AnyDesk](AnyDesk.md), etc.) protect data with end-to-end encryption and multi-factor authentication. However, selecting vendors with good reputations is important.

**Q: What if the user doesn't want their operations monitored?**

A: Users can disconnect at any time during a session. Additionally, support representatives must clearly explain "what operations they will perform."

**Q: What if the connection fails?**

A: Most services provide fallback options like phone or chat support.
