営業時間外やエージェント離席時など、 ライブチャットを対応可能なエージェントが不在の場合、 顧客はチャットボタンからメッセージを残すことができます。

### メッセージ機能を設定する

チャットボタンがオフラインに表示されるメッセージ機能の設定手順は、 「[チャットボタンの設定方法](http://support.smartweb.jp/liveagent/717046-%E3%83%81%E3%83%A3%E3%83%83%E3%83%88%E3%83%9C%E3%82%BF%E3%83%B3%E3%81%AE%E8%A8%AD%E5%AE%9A%E6%96%B9%E6%B3%95#anc3)」 を参照ください。

#### チャットのオフライン時に、「メッセージ機能」を無効にするには

エージェント不在時のメッセージ機能を解除するには、エージェント画面左メニューより 設定＞チャット＞チャットボタン＞編集＞オフラインボタン を選択し、「オフラインボタンを利用する」のチェックをオフにしてください。オフライン時にチャットボタンが非表示となります。
