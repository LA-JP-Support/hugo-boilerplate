JavaScript の実行が有効となっているブラウザであれば、スマートフォン、タブレットのような モバイル機器を利用中の顧客も、チャットサポートの依頼をすることが可能です

![](/liveagent/scripts/file.php?view=Y&file=2b67b0466765e073d25a242bfba87548)

スマートフォンのブラウザでチャットボタンをタップすると、モバイル機器の画面サイズに最適化されたチャットウィンドウが新たに立ち上がります。メッセージアプリを利用する感覚で、LiveAgent のエージェントとリアルタイムのコミュニケーションを図ることができます。

顧客側からチャットを終了する際は、チャットウィンドウ右上のメニューをタップし、「チャットの終了」をタップしてください。
