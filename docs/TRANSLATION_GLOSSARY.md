# Translation Glossary (英日翻訳用語集)

## Overview
This glossary ensures consistent translation of technical terms from English to Japanese across all content. Use this as a reference when translating blog posts, documentation, and UI text.

## Usage Instructions

### For Human Translators
1. Search this glossary before translating any technical term
2. Use the exact Japanese translation provided
3. If a term is not in the glossary, add it after translation

### For AI Translation APIs
Include this glossary in your translation prompt:

```
Use the following translation glossary for consistent terminology:
[PASTE RELEVANT TERMS FROM GLOSSARY]

Translate the content while strictly following these term translations.
```

## Translation Style Guidelines

### Particles with Bold Text
When a proper noun or technical term is bold in English, keep only the term bold in Japanese, excluding particles:

```markdown
English: **Anthropic**'s Claude
Japanese: **Anthropic**のClaude (NOT: **Anthropicの**Claude)

English: **Google**'s Gemini
Japanese: **Google**のGemini (NOT: **Googleの**Gemini)
```

### Consistency in Katakana Length
Use long vowel marks (ー) consistently:

✅ CORRECT:
- サーバー (server)
- ユーザー (user)
- コンピューター (computer)

❌ WRONG:
- サーバ
- ユーザ
- コンピュータ

### Mixed Kanji-Katakana Terms
Some terms use both kanji and katakana:

- 事前トレーニング (pre-training)
- 自然言語処理 (natural language processing)
- コンテンツ管理 (content management)
